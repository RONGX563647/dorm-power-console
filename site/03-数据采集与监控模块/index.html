    二、需求分析与设计¶

    2.1 业务场景¶

    场景 1：设备数据上报

    智能插座 → MQTT Broker → MqttBridge → 数据验证 → 存储到 Telemetry 表 → WebSocket 推送 → 前端实时更新

    场景 2：历史数据查询

    用户选择设备和时间范围 → 后端查询 Telemetry 表 → 分页返回 → 前端渲染图表

    场景 3：实时监控

    设备上报数据 → WebSocket 推送 → 前端订阅 → 实时更新图表（1 秒刷新）

    场景 4：数据导出

    用户选择导出范围 → 后端生成 CSV 文件 → 返回下载链接 → 用户下载

    2.2 功能需求清单¶

    功能 描述 优先级
    遥测数据接收 通过 MQTT 接收设备上报数据 P0
    遥测数据存储 保存到数据库，支持高频写入 P0
    实时数据推送 WebSocket 推送实时数据 P0
    历史数据查询 按时间范围、设备查询 P0
    数据导出 导出 CSV/Excel 格式 P1
    实时监控 1 秒刷新实时数据 P1
    系统指标采集 CPU、内存、设备在线率 P2
    定时任务管理 数据清理等定时任务 P2

    2.3 技术选型与理由¶

    技术 说明 版本 选型理由
    MQTT 设备数据上报协议 3.1.1 ✅ 发布/订阅模式：设备发布数据，服务端订阅消费
    ✅ 主题过滤：支持通配符主题（dorm/telemetry/#）
    ✅ QoS 保障：关键数据使用 QoS 1 确保到达
    ✅ 低延迟：适合高频实时数据传输
    WebSocket 实时数据推送 - ✅ 全双工通信：服务端主动推送数据到前端
    ✅ 低延迟：一次握手，持续通信
    ✅ 轻量协议：相比 HTTP 轮询，减少带宽消耗
    ✅ Spring 支持：Spring WebSocket 简化开发
    Spring Scheduler 定时任务调度 3.2.x ✅ 内置支持：Spring Boot 内置，无需额外依赖
    ✅ Cron 表达式：灵活的调度策略
    ✅ 异步执行：支持@Async 异步任务
    ⚠️ 局限性：单机调度，分布式场景需用 Quartz/XXL-JOB
    ECharts 图表库 5.x ✅ 丰富图表：支持折线图、柱状图、仪表盘等
    ✅ 大数据量：支持 canvas 渲染，性能优秀
    ✅ 响应式：自适应不同屏幕尺寸
    ✅ 中文文档：百度开源，文档完善

    技术对比与取舍： - WebSocket vs SSE：选择 WebSocket 因为需要双向通信（前端也可发送控制命令） - ECharts vs Chart.js：选择 ECharts 因为功能更丰富，适合复杂的数据可视化场景 - 定时任务 vs 实时计算：教学场景用定时任务简化实现，生产环境考虑 Flink 实时计算 | PostgreSQL | 时序数据存储 | 14+ |

    package com.dormpower.model;

import jakarta.persistence.*;
import lombok.Data;
import java.math.BigDecimal;
import java.time.LocalDateTime;
import java.util.UUID;

@Data
@Entity
@Table(name = "telemetry", indexes = {
    @Index(name = "idx_telemetry_device_time", columnList = "deviceId, timestamp"),
    @Index(name = "idx_telemetry_timestamp", columnList = "timestamp")
})
public class Telemetry {

    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;

    @Column(name = "device_id", nullable = false)
    private UUID deviceId;

    @Column(precision = 10, scale = 2)
    private BigDecimal voltage; // 电压（V）

    @Column(precision = 10, scale = 3)
    private BigDecimal current; // 电流（A）

    @Column(precision = 10, scale = 2)
    private BigDecimal power; // 功率（W）

    @Column(precision = 5, scale = 3)
    private BigDecimal powerFactor; // 功率因数

    @Column(precision = 10, scale = 2)
    private BigDecimal frequency; // 频率（Hz）

    @Column(precision = 12, scale = 4)
    private BigDecimal activeEnergy; // 有功电能（kWh）

    @Column(precision = 12, scale = 4)
    private BigDecimal reactiveEnergy; // 无功电能（kVarh）

    @Column(precision = 5, scale = 2)
    private BigDecimal temperature; // 温度（℃）

    @Column(precision = 5, scale = 2)
    private BigDecimal humidity; // 湿度（%）

    @Column(nullable = false)
    private LocalDateTime timestamp; // 数据采集时间

    private LocalDateTime createdAt = LocalDateTime.now();

    private Integer quality; // 数据质量标识
}

    package com.dormpower.repository;

import com.dormpower.model.Telemetry;
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.data.jpa.repository.JpaRepository;
import org.springframework.data.jpa.repository.Query;
import org.springframework.data.repository.query.Param;
import org.springframework.stereotype.Repository;

import java.time.LocalDateTime;
import java.util.List;
import java.util.Optional;
import java.util.UUID;

@Repository
public interface TelemetryRepository extends JpaRepository<Telemetry, UUID> {

    Page<Telemetry> findByDeviceIdAndTimestampBetween(
        UUID deviceId,
        LocalDateTime startTime,
        LocalDateTime endTime,
        Pageable pageable
    );

    List<Telemetry> findByDeviceIdAndTimestampBetweenOrderByTimestampDesc(
        UUID deviceId,
        LocalDateTime startTime,
        LocalDateTime endTime
    );

    Optional<Telemetry> findTopByDeviceIdOrderByTimestampDesc(UUID deviceId);

    @Query("SELECT t FROM Telemetry t WHERE t.deviceId IN :deviceIds AND t.timestamp BETWEEN :startTime AND :endTime")
    List<Telemetry> findByDeviceIdsAndTimestampBetween(
        @Param("deviceIds") List<UUID> deviceIds,
        @Param("startTime") LocalDateTime startTime,
        @Param("endTime") LocalDateTime endTime
    );

    @Query("SELECT AVG(t.power) FROM Telemetry t WHERE t.deviceId = :deviceId AND t.timestamp BETWEEN :startTime AND :endTime")
    Optional<Double> findAveragePower(
        @Param("deviceId") UUID deviceId,
        @Param("startTime") LocalDateTime startTime,
        @Param("endTime") LocalDateTime endTime
    );
}

    3.2 创建 TelemetryServiceImpl¶

    文件位置: backend/src/main/java/com/dormpower/service/impl/TelemetryServiceImpl.java

    package com.dormpower.service.impl;

import com.dormpower.dto.TelemetryData;
import com.dormpower.model.Telemetry;
import com.dormpower.repository.TelemetryRepository;
import com.dormpower.service.TelemetryService;
import com.dormpower.websocket.WebSocketManager;
import lombok.RequiredArgsConstructor;
import org.modelmapper.ModelMapper;
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.time.LocalDateTime;
import java.util.List;
import java.util.Optional;
import java.util.UUID;

@Service
@RequiredArgsConstructor
public class TelemetryServiceImpl implements TelemetryService {

    private final TelemetryRepository telemetryRepository;
    private final ModelMapper modelMapper;
    private final WebSocketManager webSocketManager;

    @Override
    @Transactional
    public void saveTelemetry(TelemetryData data) {
        Telemetry telemetry = modelMapper.map(data, Telemetry.class);
        telemetry.setTimestamp(LocalDateTime.now());
        telemetryRepository.save(telemetry);

        // 通过 WebSocket 推送实时数据
        webSocketManager.sendTelemetryUpdate(telemetry);
    }

    @Override
    public Page<Telemetry> getTelemetryByDevice(
        UUID deviceId,
        LocalDateTime startTime,
        LocalDateTime endTime,
        Pageable pageable
    ) {
        return telemetryRepository.findByDeviceIdAndTimestampBetween(
            deviceId, startTime, endTime, pageable
        );
    }

    @Override
    public Optional<Telemetry> getLatestTelemetry(UUID deviceId) {
        return telemetryRepository.findTopByDeviceIdOrderByTimestampDesc(deviceId);
    }

    @Override
    public List<Telemetry> getBatchTelemetry(
        List<UUID> deviceIds,
        LocalDateTime startTime,
        LocalDateTime endTime
    ) {
        return telemetryRepository.findByDeviceIdsAndTimestampBetween(
            deviceIds, startTime, endTime
        );
    }

    @Override
    public Optional<Double> getAveragePower(UUID deviceId, LocalDateTime startTime, LocalDateTime endTime) {
        return telemetryRepository.findAveragePower(deviceId, startTime, endTime);
    }
}

    package com.dormpower.controller;

import com.dormpower.model.Telemetry;
import com.dormpower.service.TelemetryService;
import lombok.RequiredArgsConstructor;
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.format.annotation.DateTimeFormat;
import org.springframework.http.ResponseEntity;
import org.springframework.security.access.prepost.PreAuthorize;
import org.springframework.web.bind.annotation.*;

import java.time.LocalDateTime;
import java.util.List;
import java.util.Optional;
import java.util.UUID;

@RestController
@RequestMapping("/api/telemetry")
@RequiredArgsConstructor
public class TelemetryController {

    private final TelemetryService telemetryService;

    @GetMapping
    @PreAuthorize("hasAnyAuthority('device:read', 'admin')")
    public ResponseEntity<Page<Telemetry>> getTelemetry(
        @RequestParam UUID deviceId,
        @RequestParam @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) LocalDateTime startTime,
        @RequestParam @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) LocalDateTime endTime,
        Pageable pageable
    ) {
        return ResponseEntity.ok(
            telemetryService.getTelemetryByDevice(deviceId, startTime, endTime, pageable)
        );
    }

    @GetMapping("/latest")
    @PreAuthorize("hasAnyAuthority('device:read', 'admin')")
    public ResponseEntity<Telemetry> getLatestTelemetry(@RequestParam UUID deviceId) {
        return telemetryService.getLatestTelemetry(deviceId)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }

    @PostMapping("/batch")
    @PreAuthorize("hasAnyAuthority('device:read', 'admin')")
    public ResponseEntity<List<Telemetry>> getBatchTelemetry(
        @RequestBody BatchTelemetryRequest request
    ) {
        List<Telemetry> telemetry = telemetryService.getBatchTelemetry(
            request.getDeviceIds(),
            request.getStartTime(),
            request.getEndTime()
        );
        return ResponseEntity.ok(telemetry);
    }

    @GetMapping("/average-power")
    @PreAuthorize("hasAnyAuthority('device:read', 'admin')")
    public ResponseEntity<Double> getAveragePower(
        @RequestParam UUID deviceId,
        @RequestParam @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) LocalDateTime startTime,
        @RequestParam @DateTimeFormat(iso = DateTimeFormat.ISO.DATE_TIME) LocalDateTime endTime
    ) {
        return telemetryService.getAveragePower(deviceId, startTime, endTime)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }

    @Record
    public static class BatchTelemetryRequest {
        private List<UUID> deviceIds;
        private LocalDateTime startTime;
        private LocalDateTime endTime;

        // Getters and Setters
        public List<UUID> getDeviceIds() { return deviceIds; }
        public void setDeviceIds(List<UUID> deviceIds) { this.deviceIds = deviceIds; }
        public LocalDateTime getStartTime() { return startTime; }
        public void setStartTime(LocalDateTime startTime) { this.startTime = startTime; }
        public LocalDateTime getEndTime() { return endTime; }
        public void setEndTime(LocalDateTime endTime) { this.endTime = endTime; }
    }
}

  // 获取平均功率
  async getAveragePower(
    deviceId: string,
    startTime: string,
    endTime: string

      startTime: queryForm.startTime,
      endTime: queryForm.endTime,
      page: currentPage.value - 1,
      size: pageSize.value

    telemetryData.value = response.content || [];
    total.value = response.totalElements || 0;

    updateChart();

  loadTelemetry();

      smooth: true
    }]

onMounted(() => {

    问题 1：大量数据写入导致数据库压力大¶

    现象: 数据写入慢，CPU 占用高

    解决方案:

    // 使用批量插入
@Transactional
public void batchSaveTelemetry(List<TelemetryData> dataList) {
    List<Telemetry> telemetryList = dataList.stream()
        .map(data -> modelMapper.map(data, Telemetry.class))
        .collect(Collectors.toList());

    // 分批保存，每 1000 条一批
    int batchSize = 1000;
    for (int i = 0; i < telemetryList.size(); i += batchSize) {
        int end = Math.min(i + batchSize, telemetryList.size());
        telemetryRepository.saveAll(telemetryList.subList(i, end));
    }
}

    问题 3：图表数据点过多渲染慢¶

    现象: 图表卡顿，浏览器 CPU 占用高

    解决方案:

    // 数据降采样
function downsample(data: number[], threshold: number) {
    if (data.length <= threshold) return data;

    const sampled: number[] = [];
    const step = Math.floor(data.length / threshold);

    for (let i = 0; i < data.length; i += step) {
        sampled.push(data[i]);
    }

    return sampled;
}
