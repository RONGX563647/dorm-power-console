# 后端API接口文档

## 1. 接口概述

本文档描述了宿舍电源管理系统后端的所有RESTful API接口，包括认证、设备管理、命令控制、遥测数据、计费管理、系统监控等功能。

### 1.1 基础信息
- **API基础路径**: `http://localhost:8000`
- **认证方式**: JWT Bearer Token
- **响应格式**: JSON
- **编码**: UTF-8

### 1.2 认证说明
- 除健康检查接口外，所有接口都需要在请求头中添加 `Authorization: Bearer <token>`
- 获取token的方式：通过登录接口 `/api/auth/login`

### 1.3 错误响应格式
```json
{
  "ok": false,
  "code": "ERROR_CODE",
  "message": "错误信息"
}
```

## 2. 接口分类

- **路径**: `/health`

- **描述**: 检查服务是否正常运行
- **响应示例**:
  ```json
  {
    "status": "UP",
    "service": "dorm-power-backend",
    "version": "1.0.0"
  ```

## 4. 接口使用示例

### 4.1 登录获取token
```bash
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"account": "admin", "password": "123456"}'
```

### 4.2 获取设备列表
```bash
curl -X GET http://localhost:8000/api/devices \
  -H "Authorization: Bearer <token>"
```

### 4.3 控制设备
```bash
curl -X POST http://localhost:8000/api/strips/device_001/cmd \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"action": "toggle", "socket": 1, "value": true}'
```

### 4.4 获取遥测数据
```bash
curl -X GET "http://localhost:8000/api/telemetry?device=device_001&range=24h" \
  -H "Authorization: Bearer <token>"
```

### 4.5 生成账单
```bash
curl -X POST "http://localhost:8000/api/billing/bills/generate?roomId=room_001&period=2024-01" \
  -H "Authorization: Bearer <token>"
```

## 5. 注意事项

1. **认证**: 所有接口（除健康检查外）都需要JWT认证
2. **速率限制**: 部分接口有速率限制，如登录、注册等
3. **权限**: 部分接口需要特定角色权限
4. **参数验证**: 所有接口都有参数验证，请确保参数格式正确
5. **错误处理**: 接口返回错误时，请根据错误码和消息进行处理
6. **数据安全**: 敏感操作会记录审计日志
