## 一、系统架构设计

### 1.1 并发挑战

宿舍用电管理系统需要支持 **10,000+ 设备同时在线**，面临以下挑战：

- **连接管理**: 维持大量 MQTT 长连接
- **消息处理**: 高频设备数据上报
- **实时推送**: 向多个客户端推送状态
- **数据写入**: 高频时序数据入库

### 1.2 架构演进

**阶段 1：单体应用**
```
客户端 → Spring Boot → 数据库
```

**阶段 2：引入消息队列**
```
设备 → MQTT → RabbitMQ → Spring Boot → 数据库
```

**阶段 3：微服务拆分**
```
设备网关服务 + 设备管理服务 + 计费服务 + 告警服务
```

### 2.1 连接池管理

**问题**: 传统 HTTP 连接池无法支持 10,000+ 长连接

**解决方案**:

```java
// MQTT 连接池
@Component
public class MqttConnectionPool {
    
    private final Map<String, MqttClient> clientPool = new ConcurrentHashMap<>();
    private final Semaphore semaphore;
    
    public MqttConnectionPool(@Value("${mqtt.max-connections:10000}") int maxConnections) {
        this.semaphore = new Semaphore(maxConnections);
    }
    
    public MqttClient acquire(String clientId) throws Exception {
        semaphore.acquire();
        return clientPool.computeIfAbsent(clientId, this::createClient);
    }
    
    public void release(String clientId) {
        clientPool.remove(clientId);
        semaphore.release();
    }
}
```

### 2.3 多级缓存

**Caffeine + Redis 两级缓存**:

```java
@Service
public class DeviceCacheService {
    
    private final Cache<String, DeviceDTO> localCache = Caffeine.newBuilder()
        .maximumSize(1000)
        .expireAfterWrite(5, TimeUnit.MINUTES)
        .build();
    
    private final RedisTemplate<String, DeviceDTO> redisTemplate;
    
    public DeviceDTO getDevice(String deviceId) {
        // 1. 本地缓存
        DeviceDTO device = localCache.getIfPresent(deviceId);
        if (device != null) {
            return device;
        }
        
        // 2. Redis 缓存
        device = redisTemplate.opsForValue().get("device:" + deviceId);
        if (device != null) {
            localCache.put(deviceId, device);
            return device;
        }
        
        // 3. 数据库查询
        device = deviceRepository.findById(deviceId);
        if (device != null) {
            redisTemplate.opsForValue().set("device:" + deviceId, device, 10, TimeUnit.MINUTES);
            localCache.put(deviceId, device);
        }
        
        return device;
    }
}
```

### 3.2 批量操作

**批量插入遥测数据**:

```java
@Service
public class TelemetryBatchService {
    
    private final JdbcTemplate jdbcTemplate;
    private static final int BATCH_SIZE = 1000;
    
    @Transactional
    public void batchInsert(List<TelemetryData> dataList) {
        List<List<TelemetryData>> partitions = Lists.partition(dataList, BATCH_SIZE);
        
        for (List<TelemetryData> partition : partitions) {
            jdbcTemplate.batchUpdate(
                "INSERT INTO telemetry_data (id, device_id, data, created_at) VALUES (?, ?, ?, ?)",
                new BatchPreparedStatementSetter() {
                    @Override
                    public void setValues(PreparedStatement ps, int i) throws SQLException {
                        TelemetryData data = partition.get(i);
                        ps.setObject(1, data.getId());
                        ps.setObject(2, data.getDeviceId());
                        ps.setString(3, data.getDataJson());
                        ps.setTimestamp(4, Timestamp.valueOf(data.getCreatedAt()));
                    }
                    
                    @Override
                    public int getBatchSize() {
                        return partition.size();
                    }
                }
            );
        }
    }
}
```

### 3.3 连接优化

**WebSocket 连接管理**:

```java
@Component
public class WebSocketManager {
    
    private final Map<String, WebSocketSession> sessions = new ConcurrentHashMap<>();
    private final Map<String, Set<String>> subscriptions = new ConcurrentHashMap<>();
    
    public void addSession(String sessionId, WebSocketSession session) {
        sessions.put(sessionId, session);
    }
    
    public void subscribe(String sessionId, String topic) {
        subscriptions.computeIfAbsent(topic, k -> new CopyOnWriteArraySet<>())
                     .add(sessionId);
    }
    
    public void publish(String topic, Object message) {
        Set<String> subscribers = subscriptions.get(topic);
        if (subscribers != null) {
            String jsonMessage = toJson(message);
            subscribers.forEach(sessionId -> {
                WebSocketSession session = sessions.get(sessionId);
                if (session != null && session.isOpen()) {
                    try {
                        session.sendMessage(new TextMessage(jsonMessage));
                    } catch (IOException e) {
                        log.error("消息发送失败", e);
                    }
                }
            });
        }
    }
}
```

## 五、安全性

### 5.1 限流保护

**Redis 滑动窗口限流**:

```java
@Component
public class RateLimiterService {
    
    private final RedisTemplate<String, String> redisTemplate;
    
    public boolean tryAcquire(String key, int limit, long windowSeconds) {
        String redisKey = "rate_limit:" + key;
        long now = System.currentTimeMillis();
        long windowStart = now - windowSeconds * 1000;
        
        String script = """
            local key = KEYS[1]
            local window_start = tonumber(ARGV[1])
            local window_end = tonumber(ARGV[2])
            local limit = tonumber(ARGV[3])
            
            -- 移除过期数据
            redis.call('ZREMRANGEBYSCORE', key, 0, window_start)
            
            -- 获取当前窗口内的请求数
            local count = redis.call('ZCARD', key)
            
            if count < limit then
                redis.call('ZADD', key, window_end, tostring(math.random()))
                redis.call('EXPIRE', key, window_seconds)
                return 1
            else
                return 0
            end
            """;
        
        Long result = redisTemplate.execute(
            new DefaultRedisScript<>(script, Long.class),
            Collections.singletonList(redisKey),
            String.valueOf(windowStart),
            String.valueOf(now),
            String.valueOf(limit)
        );
        
        return result != null && result == 1;
    }
}
```

### 5.2 数据加密

**敏感数据加密**:

```java
@Component
public class DataEncryptionService {
    
    private static final String ALGORITHM = "AES/GCM/NoPadding";
    private static final int GCM_IV_LENGTH = 12;
    private static final int GCM_TAG_LENGTH = 128;
    
    @Value("${encryption.key}")
    private String encryptionKey;
    
    public String encrypt(String plaintext) throws Exception {
        SecretKeySpec keySpec = new SecretKeySpec(
            Base64.getDecoder().decode(encryptionKey), "AES");
        
        Cipher cipher = Cipher.getInstance(ALGORITHM);
        byte[] iv = new byte[GCM_IV_LENGTH];
        SecureRandom random = new SecureRandom();
        random.nextBytes(iv);
        
        GCMParameterSpec parameterSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
        cipher.init(Cipher.ENCRYPT_MODE, keySpec, parameterSpec);
        
        byte[] ciphertext = cipher.doFinal(plaintext.getBytes(StandardCharsets.UTF_8));
        
        byte[] encrypted = new byte[GCM_IV_LENGTH + ciphertext.length];
        System.arraycopy(iv, 0, encrypted, 0, GCM_IV_LENGTH);
        System.arraycopy(ciphertext, 0, encrypted, GCM_IV_LENGTH, ciphertext.length);
        
        return Base64.getEncoder().encodeToString(encrypted);
    }
    
    public String decrypt(String encryptedText) throws Exception {
        byte[] encrypted = Base64.getDecoder().decode(encryptedText);
        
        SecretKeySpec keySpec = new SecretKeySpec(
            Base64.getDecoder().decode(encryptionKey), "AES");
        
        byte[] iv = new byte[GCM_IV_LENGTH];
        System.arraycopy(encrypted, 0, iv, 0, GCM_IV_LENGTH);
        
        byte[] ciphertext = new byte[encrypted.length - GCM_IV_LENGTH];
        System.arraycopy(encrypted, GCM_IV_LENGTH, ciphertext, 0, ciphertext.length);
        
        Cipher cipher = Cipher.getInstance(ALGORITHM);
        GCMParameterSpec parameterSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
        cipher.init(Cipher.DECRYPT_MODE, keySpec, parameterSpec);
        
        byte[] plaintext = cipher.doFinal(ciphertext);
        return new String(plaintext, StandardCharsets.UTF_8);
    }
}
```
