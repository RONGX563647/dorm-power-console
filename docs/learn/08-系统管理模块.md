## 二、需求分析与设计

### 2.1 业务场景

**场景 1：系统配置管理**
```
管理员修改配置 → 配置保存 → 缓存更新 → 配置生效
```

**场景 2：审计日志记录**
```
用户执行操作 → AOP 切面拦截 → 记录操作日志 → 异步保存
```

**场景 3：IP 访问控制**
```
用户访问系统 → IP 检查 → 白名单/黑名单验证 → 允许/拒绝访问
```

### 2.3 技术选型与理由

| 技术 | 说明 | 版本 | 选型理由 |
|------|------|------|----------|
| **Spring AOP** | 面向切面编程 | 3.2.x | ✅ **审计日志**：通过切面自动记录操作日志<br>✅ **解耦业务**：日志逻辑与业务逻辑分离<br>✅ **注解驱动**：@AuditLog 自定义注解标记需要审计的方法<br>✅ **性能监控**：可记录方法执行时间，分析性能瓶颈 |
| **Caffeine** | 本地缓存 | 3.x | ✅ **高性能**：基于 Window-TinyLfu 算法，命中率高于 Guava<br>✅ **内存安全**：支持基于大小、时间、引用强度淘汰<br>✅ **Spring 集成**：@Cacheable 注解简化缓存配置<br>⚠️ **局限性**：单机缓存，分布式场景需用 Redis |
| **Thymeleaf** | 模板引擎 | 3.x | ✅ **日志查看**：服务端渲染日志页面<br>✅ **Spring 集成**：与 Spring MVC 无缝集成<br>⚠️ **替代方案**：前后端分离架构中，更多使用 Vue/React |
| **Logback** | 日志框架 | 1.4.x | ✅ **Spring 默认**：Spring Boot 默认日志框架<br>✅ **配置灵活**：支持 XML/Groovy 配置<br>✅ **异步日志**：AsyncAppender 提升性能<br>✅ **滚动策略**：支持按时间/大小滚动日志文件 |

**技术对比与取舍**：
- **AOP vs 手动日志**：选择 AOP 因为日志是横切关注点，适合切面处理
- **本地缓存 vs Redis**：简单配置用 Caffeine，共享数据用 Redis
- **Logback vs Log4j2**：选择 Logback 因为 Spring Boot 默认支持，配置简单

```java
// SystemConfig.java
package com.dormpower.model;

import jakarta.persistence.*;
import lombok.Data;
import java.time.LocalDateTime;
import java.util.UUID;

@Data
@Entity
@Table(name = "system_config", uniqueConstraints = {
    @UniqueConstraint(columnNames = "config_key")
})
public class SystemConfig {
    
    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;
    
    @Column(name = "config_key", nullable = false, length = 100)
    private String configKey;
    
    @Column(name = "config_value", nullable = false, length = 2000)
    private String configValue;
    
    @Enumerated(EnumType.STRING)
    @Column(nullable = false)
    private ConfigType configType = ConfigType.STRING;
    
    @Column(length = 50)
    private String category;
    
    @Column(length = 500)
    private String description;
    
    @Column(nullable = false)
    private boolean isPublic = false;
    
    @Column(nullable = false)
    private boolean isEditable = true;
    
    private LocalDateTime updatedAt = LocalDateTime.now();
    
    @Column(name = "updated_by")
    private UUID updatedBy;
}

enum ConfigType {
    STRING, NUMBER, BOOLEAN, JSON
}
```

```java
// AuditLog.java
package com.dormpower.model;

import jakarta.persistence.*;
import lombok.Data;
import java.time.LocalDateTime;
import java.util.UUID;

@Data
@Entity
@Table(name = "audit_log", indexes = {
    @Index(name = "idx_audit_user", columnList = "userId"),
    @Index(name = "idx_audit_action", columnList = "action"),
    @Index(name = "idx_audit_time", columnList = "executedAt")
})
public class AuditLog {
    
    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;
    
    @Column(name = "user_id")
    private UUID userId;
    
    @Column(length = 100)
    private String username;
    
    @Column(nullable = false, length = 100)
    private String action;
    
    @Column(length = 100)
    private String resourceType;
    
    @Column(name = "resource_id")
    private UUID resourceId;
    
    @Column(length = 50)
    private String operation;
    
    @Column(length = 20)
    private String requestMethod;
    
    @Column(length = 500)
    private String requestUrl;
    
    @Column(columnDefinition = "TEXT")
    private String requestParams;
    
    private Integer responseStatus;
    
    private Long responseTime; // ms
    
    @Column(length = 50)
    private String ipAddress;
    
    @Column(length = 500)
    private String userAgent;
    
    @Column(nullable = false)
    private LocalDateTime executedAt = LocalDateTime.now();
    
    @Column(nullable = false)
    private LocalDateTime createdAt = LocalDateTime.now();
    
    @Column(length = 500)
    private String remark;
}
```

### 步骤 4：Controller 层

#### 4.1 创建 SystemConfigController

**文件位置**: `backend/src/main/java/com/dormpower/controller/SystemConfigController.java`

```java
package com.dormpower.controller;

import com.dormpower.model.SystemConfig;
import com.dormpower.service.SystemConfigService;
import lombok.RequiredArgsConstructor;
import org.springframework.http.ResponseEntity;
import org.springframework.security.access.prepost.PreAuthorize;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/system-configs")
@RequiredArgsConstructor
public class SystemConfigController {
    
    private final SystemConfigService systemConfigService;
    
    @GetMapping
    @PreAuthorize("hasAnyAuthority('system:read', 'admin')")
    public ResponseEntity<List<SystemConfig>> getConfigs(
        @RequestParam(required = false) String category
    ) {
        if (category != null) {
            return ResponseEntity.ok(systemConfigService.getConfigsByCategory(category));
        }
        return ResponseEntity.ok((List) systemConfigService.getAllConfigs().entrySet().stream()
            .map(e -> {
                SystemConfig config = new SystemConfig();
                config.setConfigKey(e.getKey());
                config.setConfigValue(e.getValue());
                return config;
            }).toList());
    }
    
    @GetMapping("/{key}")
    @PreAuthorize("hasAnyAuthority('system:read', 'admin')")
    public ResponseEntity<SystemConfig> getConfig(@PathVariable String key) {
        return ResponseEntity.ok(systemConfigService.getConfig(key));
    }
    
    @PutMapping("/{key}")
    @PreAuthorize("hasAnyAuthority('system:write', 'admin')")
    public ResponseEntity<SystemConfig> updateConfig(
        @PathVariable String key,
        @RequestBody Map<String, String> body
    ) {
        return ResponseEntity.ok(
            systemConfigService.setConfig(key, body.get("value"), body.get("category"))
        );
    }
}
```

### 4.2 API 封装

**文件位置**: `frontend/src/api/system.ts`

```typescript
import axios from 'axios';
import type { SystemConfig, AuditLog } from '@/types/system';

const CONFIG_API_BASE = '/api/system-configs';
const AUDIT_API_BASE = '/api/audit-logs';

export const systemApi = {
  async getConfigs(category?: string): Promise<SystemConfig[]> {
    const response = await axios.get(CONFIG_API_BASE, {
      params: { category }
    });
    return response.data;
  },
  
  async updateConfig(key: string, value: string, category?: string) {
    const response = await axios.put(`${CONFIG_API_BASE}/${key}`, {
      value,
      category
    });
    return response.data;
  }
};

export const auditApi = {
  async getAuditLogs(
    userId?: string,
    action?: string,
    startTime?: string,
    endTime?: string,
    page = 0,
    size = 20
  ) {
    const response = await axios.get(AUDIT_API_BASE, {
      params: { userId, action, startTime, endTime, page, size }
    });
    return response.data;
  }
};
```
