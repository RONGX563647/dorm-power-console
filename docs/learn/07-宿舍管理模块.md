## 二、需求分析与设计

### 2.1 业务场景

**场景 1：楼栋管理**
```
创建楼栋 → 设置楼层数 → 设置房间数 → 楼栋信息维护
```

**场景 2：房间管理**
```
创建房间 → 设置床位 → 分配设备 → 房间信息维护
```

**场景 3：入住管理**
```
学生申请 → 管理员审核 → 分配房间 → 办理入住 → 设备绑定
```

**场景 4：退宿管理**
```
学生申请 → 管理员审核 → 检查设备 → 办理退宿 → 清理数据
```

### 2.2 功能需求清单

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 楼栋管理 | CRUD 操作 | P0 |
| 房间管理 | CRUD 操作 | P0 |
| 入住管理 | 入住办理 | P0 |
| 退宿管理 | 退宿办理 | P0 |
| 住宿统计 | 入住率统计 | P1 |
| 设备绑定 | 房间设备关联 | P2 |

### 2.3 技术选型与理由

| 技术 | 说明 | 版本 | 选型理由 |
|------|------|------|----------|
| **JPA/Hibernate** | ORM 框架 | 3.2.x | ✅ **级联操作**：楼栋 - 房间 - 学生层级关系自动管理<br>✅ **查询简化**：Spring Data JPA 方法命名查询<br>✅ **事务管理**：@Transactional 保证数据一致性 |
| **Spring Data REST** | RESTful API | 3.2.x | ✅ **快速开发**：Repository 自动暴露为 REST API<br>✅ **HATEOAS**：支持资源关联链接<br>⚠️ **简化实现**：教学场景手动编写 Controller 更清晰 |
| **Vue Element Plus** | UI 组件库 | 2.x | ✅ **表格组件**：ElTable 支持分页、排序、筛选<br>✅ **表单组件**：ElForm 支持验证、布局<br>✅ **对话框**：ElDialog 实现弹窗编辑 |
| **EasyExcel** | Excel 导入导出 | 3.x | ✅ **批量导入**：支持 Excel 批量导入学生信息<br>✅ **模板导出**：导出宿舍分配报表<br>✅ **流式处理**：避免大数据量 OOM |

**技术对比与取舍**：
- **单表 vs 层级设计**：选择楼栋 - 房间两层设计，符合实际业务场景
- **手动 SQL vs JPA**：选择 JPA 因为 CRUD 操作多，查询逻辑简单
- **前端分页 vs 后端分页**：选择后端分页，减少前端压力，适合大数据量

**DTO 设计原则**:

```
┌─────────────────────────────────────────────┐
│            DTO 设计最佳实践                   │
├─────────────────────────────────────────────┤
│ 1. 职责单一:                                 │
│    - Request DTO: 接收前端数据,添加验证       │
│    - Response DTO: 返回前端数据,过滤敏感信息  │
│    - Message DTO: 消息传递载体(MQTT/WebSocket)│
│                                            │
│ 2. 命名规范:                                 │
│    - 请求DTO: XxxRequest.java                │
│    - 响应DTO: XxxResponse.java               │
│    - 消息DTO: XxxMessage.java                │
│                                            │
│ 3. 验证注解:                                 │
│    - @NotNull: 不能为null                    │
│    - @NotBlank: 不能为空白字符串              │
│    - @Size: 字符串/集合大小限制               │
│    - @Pattern: 正则表达式验证                 │
│    - @DecimalMin/DecimalMax: 数值范围验证     │
│                                            │
│ 4. API文档:                                  │
│    - @Schema: OpenAPI 3.0 文档注解           │
│    - description: 字段说明                   │
│    - example: 示例值                         │
│    - required: 是否必填                      │
│                                            │
│ 5. 与Entity的区别:                           │
│    - Entity: 数据库映射,包含JPA注解           │
│    - DTO: 数据传输,包含验证注解               │
│    - 两者可以相互转换,使用Mapper或手动转换    │
│                                            │
│ 6. Builder模式:                              │
│    - 响应DTO推荐使用Builder模式               │
│    - 链式调用,代码更简洁                      │
│    - 避免构造函数参数过多                     │
└─────────────────────────────────────────────┘
```

```java
package com.dormpower.controller;

import com.dormpower.dto.RoomDTO;
import com.dormpower.model.Room;
import com.dormpower.service.RoomService;
import lombok.RequiredArgsConstructor;
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.http.ResponseEntity;
import org.springframework.security.access.prepost.PreAuthorize;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.UUID;

@RestController
@RequestMapping("/api/rooms")
@RequiredArgsConstructor
public class RoomController {
    
    private final RoomService roomService;
    
    @PostMapping
    @PreAuthorize("hasAnyAuthority('room:write', 'admin')")
    public ResponseEntity<Room> createRoom(@RequestBody RoomDTO dto) {
        return ResponseEntity.ok(roomService.createRoom(dto));
    }
    
    @GetMapping
    @PreAuthorize("hasAnyAuthority('room:read', 'admin')")
    public ResponseEntity<Page<Room>> getRooms(
        @RequestParam(required = false) UUID buildingId,
        Pageable pageable
    ) {
        return ResponseEntity.ok(roomService.getRooms(buildingId, pageable));
    }
    
    @GetMapping("/{id}")
    @PreAuthorize("hasAnyAuthority('room:read', 'admin')")
    public ResponseEntity<Room> getRoom(@PathVariable UUID id) {
        return roomService.getRoom(id)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }
    
    @PutMapping("/{id}")
    @PreAuthorize("hasAnyAuthority('room:write', 'admin')")
    public ResponseEntity<Room> updateRoom(
        @PathVariable UUID id,
        @RequestBody RoomDTO dto
    ) {
        return ResponseEntity.ok(roomService.updateRoom(id, dto));
    }
    
    @DeleteMapping("/{id}")
    @PreAuthorize("hasAnyAuthority('room:write', 'admin')")
    public ResponseEntity<Void> deleteRoom(@PathVariable UUID id) {
        roomService.deleteRoom(id);
        return ResponseEntity.noContent().build();
    }
    
    @PostMapping("/{id}/checkin")
    @PreAuthorize("hasAnyAuthority('room:write', 'admin')")
    public ResponseEntity<Room> checkIn(
        @PathVariable UUID id,
        @RequestParam UUID userId
    ) {
        return ResponseEntity.ok(roomService.checkIn(id, userId));
    }
    
    @PostMapping("/{id}/checkout")
    @PreAuthorize("hasAnyAuthority('room:write', 'admin')")
    public ResponseEntity<Room> checkOut(
        @PathVariable UUID id,
        @RequestParam UUID userId
    ) {
        return ResponseEntity.ok(roomService.checkOut(id, userId));
    }
}
```

```typescript
import axios from 'axios';
import type { Room, Building, ResidenceRecord } from '@/types/room';

const ROOM_API_BASE = '/api/rooms';
const BUILDING_API_BASE = '/api/buildings';

export const roomApi = {
  async getRooms(buildingId?: string, page = 0, size = 20) {
    const response = await axios.get(ROOM_API_BASE, {
      params: { buildingId, page, size }
    });
    return response.data;
  },
  
  async createRoom(room: Partial<Room>) {
    const response = await axios.post(ROOM_API_BASE, room);
    return response.data;
  },
  
  async updateRoom(roomId: string, room: Partial<Room>) {
    const response = await axios.put(`${ROOM_API_BASE}/${roomId}`, room);
    return response.data;
  },
  
  async deleteRoom(roomId: string) {
    await axios.delete(`${ROOM_API_BASE}/${roomId}`);
  },
  
  async checkIn(roomId: string, userId: string) {
    const response = await axios.post(`${ROOM_API_BASE}/${roomId}/checkin`, null, {
      params: { userId }
    });
    return response.data;
  },
  
  async checkOut(roomId: string, userId: string) {
    const response = await axios.post(`${ROOM_API_BASE}/${roomId}/checkout`, null, {
      params: { userId }
    });
    return response.data;
  }
};

export const buildingApi = {
  async getBuildings() {
    const response = await axios.get(BUILDING_API_BASE);
    return response.data;
  },
  
  async createBuilding(building: Partial<Building>) {
    const response = await axios.post(BUILDING_API_BASE, building);
    return response.data;
  }
};
```
