## 二、需求分析与设计

### 2.1 业务场景

**场景 1：设备注册**
```
新设备到货 → 录入系统 → 分配房间 → 设备上线 → 状态变为 ONLINE
```

**场景 2：设备监控**
```
设备定时上报数据 → MQTT Broker → MqttBridge → 更新设备状态 → WebSocket 推送前端
```

**场景 3：设备分组**
```
创建设备组 → 添加设备到组 → 批量操作组内设备 → 查看组统计
```

**场景 4：固件升级**
```
上传固件 → 选择目标设备 → 推送升级命令 → 设备下载升级 → 上报新版本
```

### 2.2 功能需求清单

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 设备 CRUD | 设备创建、查询、更新、删除 | P0 |

| 设备分组 | 设备分组管理 | P1 |

### 2.3 技术选型与理由

| 技术 | 说明 | 版本 | 选型理由 |
|------|------|------|----------|
| **MQTT (Eclipse Paho)** | IoT 通信协议 | 1.3.x | ✅ **轻量级**：协议头最小仅 2 字节，适合低带宽环境<br>✅ **发布/订阅**：解耦设备与服务端，支持一对多通信<br>✅ **QoS 保障**：支持 3 个等级（0/1/2）的消息可靠性保障<br>✅ **遗嘱消息**：设备异常离线时可发送遗嘱消息<br>✅ **低功耗**：适合电池供电的 IoT 设备 |
| **Spring Data JPA** | ORM 框架 | 3.2.x | ✅ **开发效率**：注解驱动，减少样板代码<br>✅ **数据库无关**：可轻松切换数据库<br>✅ **级联操作**：自动处理设备与关联实体关系<br>⚠️ **注意**：实际项目使用 Long/IDENTITY 而非 UUID，便于数据库自增 |
| **WebSocket** | 实时推送 | - | ✅ **双向通信**：服务端可主动推送设备状态到前端<br>✅ **低延迟**：保持长连接，避免轮询开销<br>✅ **标准协议**：浏览器原生支持，无需额外插件 |
| **Lombok** | 代码简化 | - | ✅ **减少样板**：自动生成 Getter/Setter/Constructor<br>✅ **可读性**：减少视觉干扰，聚焦业务逻辑<br>✅ **维护性**：减少手动生成代码的错误 |

**技术对比与取舍**：
- **MQTT vs HTTP**：选择 MQTT 因为 IoT 设备需要长连接、低功耗、实时推送
- **JPA vs MyBatis**：选择 JPA 因为设备管理 CRUD 操作多，JPA 开发效率更高
- **WebSocket vs SSE**：选择 WebSocket 因为需要双向通信（前端也可发送控制命令）

```java
package com.dormpower.model;

import jakarta.persistence.*;
import lombok.Data;
import java.time.LocalDateTime;
import java.util.UUID;

@Data
@Entity
@Table(name = "device")
public class Device {
    
    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;
    
    @Column(unique = true, nullable = false, length = 50)
    private String deviceId; // 设备唯一标识（如 MAC 地址）
    
    @Column(nullable = false, length = 100)
    private String name; // 设备名称
    
    private String description;
    
    @Enumerated(EnumType.STRING)
    private DeviceType type; // 设备类型
    
    @Column(length = 50)
    private String model; // 设备型号
    
    @Column(length = 50)
    private String firmwareVersion; // 固件版本
    
    @Enumerated(EnumType.STRING)
    private DeviceStatus status = DeviceStatus.OFFLINE; // 设备状态
    
    @Column(length = 50)
    private String ipAddress;
    
    @Column(length = 50)
    private String macAddress;
    
    private LocalDateTime lastOnlineAt;
    private LocalDateTime lastOfflineAt;
    
    private LocalDateTime createdAt = LocalDateTime.now();
    private LocalDateTime updatedAt = LocalDateTime.now();
    
    @Column(name = "room_id")
    private UUID roomId; // 关联房间
    
    @Column(name = "group_id")
    private UUID groupId; // 关联设备组
    
    @PreUpdate
    protected void onUpdate() {
        updatedAt = LocalDateTime.now();
    }
}

enum DeviceType {
    SOCKET,    // 智能插座
    STRIP,     // 智能排插
    METER      // 智能电表
}

enum DeviceStatus {
    ONLINE,    // 在线
    OFFLINE,   // 离线
    FAULT      // 故障
}
```

#### 3.2 DeviceServiceImpl 实现

**文件位置**: `backend/src/main/java/com/dormpower/service/impl/DeviceServiceImpl.java`

```java
package com.dormpower.service.impl;

import com.dormpower.dto.request.CreateDeviceRequest;
import com.dormpower.dto.response.DeviceResponse;
import com.dormpower.model.Device;
import com.dormpower.model.DeviceStatus;
import com.dormpower.repository.DeviceRepository;
import com.dormpower.service.DeviceService;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.List;
import java.util.UUID;
import java.util.stream.Collectors;

@Service
@RequiredArgsConstructor
public class DeviceServiceImpl implements DeviceService {
    
    private final DeviceRepository deviceRepository;
    
    @Override
    @Transactional
    public DeviceResponse createDevice(CreateDeviceRequest request) {
        // 检查设备 ID 是否重复
        if (deviceRepository.existsByDeviceId(request.getDeviceId())) {
            throw new RuntimeException("设备 ID 已存在");
        }
        
        Device device = new Device();
        device.setDeviceId(request.getDeviceId());
        device.setName(request.getName());
        device.setDescription(request.getDescription());
        device.setType(request.getType());
        device.setModel(request.getModel());
        device.setRoomId(request.getRoomId());
        device.setGroupId(request.getGroupId());
        device.setStatus(DeviceStatus.OFFLINE);
        
        Device saved = deviceRepository.save(device);
        return toResponse(saved);
    }
    
    @Override
    public DeviceResponse getDevice(UUID id) {
        Device device = deviceRepository.findById(id)
            .orElseThrow(() -> new RuntimeException("设备不存在"));
        return toResponse(device);
    }
    
    @Override
    public List<DeviceResponse> getDevices(UUID roomId, UUID groupId, DeviceStatus status) {
        List<Device> devices;
        
        if (roomId != null) {
            devices = deviceRepository.findByRoomId(roomId);
        } else if (groupId != null) {
            devices = deviceRepository.findByGroupId(groupId);
        } else if (status != null) {
            devices = deviceRepository.findByStatus(status);
        } else {
            devices = deviceRepository.findAll();
        }
        
        return devices.stream().map(this::toResponse).collect(Collectors.toList());
    }
    
    @Override
    @Transactional
    public DeviceResponse updateDevice(UUID id, CreateDeviceRequest request) {
        Device device = deviceRepository.findById(id)
            .orElseThrow(() -> new RuntimeException("设备不存在"));
        
        device.setName(request.getName());
        device.setDescription(request.getDescription());
        device.setRoomId(request.getRoomId());
        device.setGroupId(request.getGroupId());
        
        Device updated = deviceRepository.save(device);
        return toResponse(updated);
    }
    
    @Override
    @Transactional
    public void deleteDevice(UUID id) {
        deviceRepository.deleteById(id);
    }
    
    @Override
    @Transactional
    public void updateDeviceStatus(String deviceId, DeviceStatus status) {
        Device device = deviceRepository.findByDeviceId(deviceId)
            .orElseThrow(() -> new RuntimeException("设备不存在"));
        
        DeviceStatus oldStatus = device.getStatus();
        device.setStatus(status);
        
        if (status == DeviceStatus.ONLINE) {
            device.setLastOnlineAt(java.time.LocalDateTime.now());
        } else if (status == DeviceStatus.OFFLINE) {
            device.setLastOfflineAt(java.time.LocalDateTime.now());
        }
        
        deviceRepository.save(device);
        
        // TODO: 记录状态变更历史
    }
    
    private DeviceResponse toResponse(Device device) {
        DeviceResponse response = new DeviceResponse();
        response.setId(device.getId());
        response.setDeviceId(device.getDeviceId());
        response.setName(device.getName());
        response.setDescription(device.getDescription());
        response.setType(device.getType());
        response.setModel(device.getModel());
        response.setFirmwareVersion(device.getFirmwareVersion());
        response.setStatus(device.getStatus());
        response.setRoomId(device.getRoomId());
        response.setGroupId(device.getGroupId());
        response.setCreatedAt(device.getCreatedAt());
        return response;
    }
}
```

```java
package com.dormpower.controller;

import com.dormpower.dto.request.CreateDeviceRequest;
import com.dormpower.dto.response.DeviceResponse;
import com.dormpower.model.DeviceStatus;
import com.dormpower.service.DeviceService;
import jakarta.validation.Valid;
import lombok.RequiredArgsConstructor;
import org.springframework.http.ResponseEntity;
import org.springframework.security.access.prepost.PreAuthorize;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.UUID;

@RestController
@RequestMapping("/api/devices")
@RequiredArgsConstructor
public class DeviceController {
    
    private final DeviceService deviceService;
    
    @PostMapping
    @PreAuthorize("hasAuthority('device:create')")
    public ResponseEntity<DeviceResponse> createDevice(
        @Valid @RequestBody CreateDeviceRequest request
    ) {
        return ResponseEntity.ok(deviceService.createDevice(request));
    }
    
    @GetMapping("/{id}")
    @PreAuthorize("hasAuthority('device:read')")
    public ResponseEntity<DeviceResponse> getDevice(@PathVariable UUID id) {
        return ResponseEntity.ok(deviceService.getDevice(id));
    }
    
    @GetMapping
    @PreAuthorize("hasAuthority('device:read')")
    public ResponseEntity<List<DeviceResponse>> getDevices(
        @RequestParam(required = false) UUID roomId,
        @RequestParam(required = false) UUID groupId,
        @RequestParam(required = false) DeviceStatus status
    ) {
        return ResponseEntity.ok(deviceService.getDevices(roomId, groupId, status));
    }
    
    @PutMapping("/{id}")
    @PreAuthorize("hasAuthority('device:update')")
    public ResponseEntity<DeviceResponse> updateDevice(
        @PathVariable UUID id,
        @Valid @RequestBody CreateDeviceRequest request
    ) {
        return ResponseEntity.ok(deviceService.updateDevice(id, request));
    }
    
    @DeleteMapping("/{id}")
    @PreAuthorize("hasAuthority('device:delete')")
    public ResponseEntity<Void> deleteDevice(@PathVariable UUID id) {
        deviceService.deleteDevice(id);
        return ResponseEntity.ok().build();
    }
}
```

```typescript
import axios from 'axios';
import type { Device, CreateDeviceRequest, DeviceGroup } from '@/types/device';

const API_BASE = '/api/devices';

export const deviceApi = {
  // 创建设备
  async createDevice(data: CreateDeviceRequest): Promise<Device> {
    const response = await axios.post(API_BASE, data);
    return response.data;
  },
  
  // 获取设备列表
  async getDevices(params?: {
    roomId?: string;
    groupId?: string;
    status?: string;
  }): Promise<Device[]> {
    const response = await axios.get(API_BASE, { params });
    return response.data;
  },
  
  // 获取设备详情

    const response = await axios.get(`${API_BASE}/${id}`);
    return response.data;
  },
  
  // 更新设备
  async updateDevice(id: string, data: CreateDeviceRequest): Promise<Device> {
    const response = await axios.put(`${API_BASE}/${id}`, data);
    return response.data;
  },
  
  // 删除设备
  async deleteDevice(id: string): Promise<void> {
    await axios.delete(`${API_BASE}/${id}`);
  }
```

```vue
<script setup lang="ts">
import { ref, reactive, onMounted } from 'vue';
import { ElMessage, ElMessageBox } from 'element-plus';
import type { FormInstance, FormRules } from 'element-plus';
import { deviceApi, type Device, type CreateDeviceRequest } from '@/api/device';

const loading = ref(false);
const devices = ref<Device[]>([]);
const dialogVisible = ref(false);
const isEdit = ref(false);
const submitting = ref(false);

const filters = reactive({
  status: '',
  keyword: ''
});

const pagination = reactive({
  page: 1,
  size: 20,
  total: 0
});

const form = reactive<CreateDeviceRequest>({
  deviceId: '',
  name: '',
  type: 'SOCKET',
  model: '',
  description: ''
});

const rules: FormRules = {
  deviceId: [{ required: true, message: '请输入设备 ID', trigger: 'blur' }],
  name: [{ required: true, message: '请输入设备名称', trigger: 'blur' }],
  type: [{ required: true, message: '请选择设备类型', trigger: 'change' }]
};

const formRef = ref<FormInstance>();

const loadDevices = async () => {
  loading.value = true;
  try {
    const data = await deviceApi.getDevices({
      status: filters.status || undefined,
      roomId: undefined,
      groupId: undefined
    });
    devices.value = data;
    pagination.total = data.length;
  } catch (error: any) {
    ElMessage.error(error.response?.data?.message || '加载失败');
  } finally {
    loading.value = false;
  }
};

const handleCreate = () => {
  isEdit.value = false;
  Object.assign(form, {
    deviceId: '',
    name: '',
    type: 'SOCKET',
    model: '',
    description: ''
  });
  dialogVisible.value = true;
};

const handleEdit = (row: Device) => {
  isEdit.value = true;
  Object.assign(form, row);
  dialogVisible.value = true;
};

const handleDelete = async (row: Device) => {
  try {
    await ElMessageBox.confirm('确定要删除该设备吗？', '提示', {
      type: 'warning'
    });
    await deviceApi.deleteDevice(row.id);
    ElMessage.success('删除成功');
    loadDevices();
  } catch (error: any) {
    if (error !== 'cancel') {
      ElMessage.error(error.response?.data?.message || '删除失败');
    }
  }
};

const handleSubmit = async () => {
  if (!formRef.value) return;
  
  await formRef.value.validate(async (valid) => {
    if (!valid) return;
    
    submitting.value = true;
    try {
      if (isEdit.value) {
        await deviceApi.updateDevice(form.deviceId, form);
      } else {
        await deviceApi.createDevice(form);
      }
      ElMessage.success(isEdit.value ? '更新成功' : '创建成功');
      dialogVisible.value = false;
      loadDevices();
    } catch (error: any) {
      ElMessage.error(error.response?.data?.message || '操作失败');
    } finally {
      submitting.value = false;
    }
  });
};

const handleSearch = () => {
  loadDevices();
};

const getStatusType = (status: string) => {
  const types: Record<string, string> = {
    ONLINE: 'success',
    OFFLINE: 'info',
    FAULT: 'danger'
  };
  return types[status] || 'info';
};

const getStatusText = (status: string) => {
  const texts: Record<string, string> = {
    ONLINE: '在线',
    OFFLINE: '离线',
    FAULT: '故障'
  };
  return texts[status] || status;
};

onMounted(() => {
  loadDevices();
});
</script>
```

### 步骤 4：状态管理

**文件位置**: `frontend/src/stores/device.ts`

```typescript
import { defineStore } from 'pinia';
import { ref } from 'vue';
import type { Device, DeviceGroup } from '@/types/device';
import { deviceApi, deviceGroupApi } from '@/api/device';

export const useDeviceStore = defineStore('device', () => {
  const devices = ref<Device[]>([]);
  const groups = ref<DeviceGroup[]>([]);
  const loading = ref(false);
  
  // 加载设备列表
  async function loadDevices(params?: any) {
    loading.value = true;
    try {
      devices.value = await deviceApi.getDevices(params);
    } finally {
      loading.value = false;
    }
  }
  
  // 加载设备组
  async function loadGroups() {
    groups.value = await deviceGroupApi.getGroups();
  }
  
  // 添加设备
  async function addDevice(device: Device) {
    devices.value.push(device);
  }
  
  // 更新设备
  function updateDevice(id: string, updates: Partial<Device>) {
    const index = devices.value.findIndex(d => d.id === id);
    if (index !== -1) {
      devices.value[index] = { ...devices.value[index], ...updates };
    }
  }
  
  // 删除设备
  function removeDevice(id: string) {
    const index = devices.value.findIndex(d => d.id === id);
    if (index !== -1) {
      devices.value.splice(index, 1);
    }
  }
  
  return {
    devices,
    groups,
    loading,
    loadDevices,
    loadGroups,
    addDevice,
    updateDevice,
    removeDevice
  };
});
```
