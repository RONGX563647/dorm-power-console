## 一、色彩系统总览

### 设计理念
本系统采用**青绿森系节能风**配色方案，通过低饱和度青绿色调传达环保节能的品牌理念，同时结合科技蓝提升现代感，砖红色作为警示色确保用户安全操作。

---

## 二、主色调规范

### 2.1 主色：#2A7965（低饱和青绿色）

| 属性 | 值 |
|------|-----|
| 十六进制 | `#2A7965` |
| RGB | `rgb(42, 121, 101)` |
| HSL | `hsl(165, 48%, 32%)` |

#### 应用场景
- ✅ 品牌标识（Logo、应用图标）
- ✅ 主要按钮（提交、确认、登录等核心操作）
- ✅ 重要标题（页面标题、区块标题）
- ✅ 导航栏（顶部导航、TabBar选中态）
- ✅ 交互元素高亮（选中状态、激活状态）
- ✅ 渐变色主色调（与 #3D967E 搭配）

#### 设计目的
通过低饱和度青绿色调传达环保节能的品牌理念，营造清新自然的视觉感受，符合宿舍管理系统的节能定位。

## 七、中性色规范

### 7.1 文字色彩

| 用途 | 色彩值 | 对比度（白底） | WCAG等级 |
|------|--------|----------------|----------|
| 主标题 | `#2A7965` | 4.5:1 | AA |
| 正文文本 | `#2A7965` | 4.5:1 | AA |
| 次要文本 | `#66A392` | 3.2:1 | AA Large |
| 辅助文本 | `#999999` | 2.3:1 | - |
| 禁用文本 | `#CCCCCC` | 1.6:1 | - |

### 7.2 边框与分割线

| 用途 | 色彩值 |
|------|--------|
| 主要边框 | `#E8F5F0` |
| 次要边框 | `#F2F7F5` |
| 分割线 | `#E8F5F0` |

### 7.3 阴影色彩
```css

/* 浅色阴影 */
box-shadow: 0 4rpx 16rpx rgba(42, 121, 101, 0.15);
```

## 八、色彩使用禁止事项

### ❌ 禁止使用的色彩
- 避免使用高饱和度的鲜艳色彩（如正红、正绿、正蓝）
- 避免使用紫色系（#667eea、#764ba2 等）
- 避免使用纯白色（#FFFFFF）作为大面积背景
- 避免使用纯黑色（#000000）作为文本色

### ⚠️ 谨慎使用的场景
- 警示色（#E86F50）不应滥用，仅用于真正需要警示的场景
- 主色调（#2A7965）不应过度使用，保持视觉焦点
- 科技辅色（#337ECC）应与主色调协调搭配

## 九、WCAG 2.1 AA 可访问性验证

### 9.1 对比度要求
| 文本类型 | 最小对比度 |
|----------|------------|
| 普通文本（18pt以下） | 4.5:1 |
| 大文本（18pt及以上） | 3:1 |
| 图形用户界面组件 | 3:1 |

### 9.2 已验证的对比组合
| 前景色 | 背景色 | 对比度 | WCAG等级 |
|--------|--------|--------|----------|
| #2A7965 | #FFFFFF | 4.5:1 | AA |
| #2A7965 | #F2F7F5 | 4.2:1 | AA |
| #66A392 | #FFFFFF | 3.2:1 | AA Large |
| #337ECC | #FFFFFF | 4.7:1 | AA |
| #E86F50 | #FFFFFF | 4.8:1 | AA |

## 十、色彩应用示例

### 10.1 按钮样式

```vue
<style lang="scss">
/* 主按钮 */
.btn-primary {
    background: linear-gradient(135deg, #2A7965 0%, #3D967E 100%);
    color: #fff;
}

/* 次要按钮 */
.btn-secondary {
    background: linear-gradient(135deg, #337ECC 0%, #4A94D9 100%);
    color: #fff;
}

/* 警告按钮 */
.btn-warning {
    background: linear-gradient(135deg, #E86F50 0%, #F08A6E 100%);
    color: #fff;
}
</style>
```

### 10.2 卡片样式

```vue
<style lang="scss">
.card {
    background: #fff;
    border-radius: 20rpx;
    box-shadow: 0 8rpx 24rpx rgba(42, 121, 101, 0.15);
}

.card-header {
    background: linear-gradient(135deg, #2A7965 0%, #3D967E 100%);
    color: #fff;
}
</style>
```

### 10.3 文本样式

```vue
<style lang="scss">
/* 标题 */
.title {
    color: #2A7965;
    font-size: 32rpx;
    font-weight: 600;
}

/* 正文 */
.body-text {
    color: #2A7965;
    font-size: 28rpx;
}

/* 次要文本 */
.secondary-text {
    color: #66A392;
    font-size: 26rpx;
}
</style>
```
