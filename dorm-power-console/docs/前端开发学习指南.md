## React基础

### 什么是React？

React是一个用于构建用户界面的JavaScript库，由Facebook（现Meta）开发。它采用组件化的方式构建UI，使代码更易于维护和复用。

### 核心概念

#### 1. 组件(Components)

组件是React应用的基本构建块。它们接受输入（称为props）并返回描述应该显示内容的React元素。

```tsx
// 函数组件示例
function Welcome({ name }: { name: string }) {
  return <h1>Hello, {name}!</h1>;
}

// 使用组件
<Welcome name="World" />
```

#### 2. JSX语法

JSX是一种JavaScript的语法扩展，允许我们在JavaScript代码中编写类似HTML的代码。

```tsx
// JSX示例
const element = <div className="greeting">Hello, world!</div>;

// 等同于
const element = React.createElement(
  'div',
  { className: 'greeting' },
  'Hello, world!'
);
```

注意：在React中使用`className`而不是`class`，因为`class`是JavaScript的保留字。

#### 3. Props（属性）

Props是组件的输入，是只读的，用于从父组件向子组件传递数据。

```tsx
// 定义Props接口
interface ButtonProps {
  text: string;
  onClick: () => void;
  disabled?: boolean; // 可选属性
}

// 使用Props
function MyButton({ text, onClick, disabled = false }: ButtonProps) {
  return (
    <button onClick={onClick} disabled={disabled}>
      {text}
    </button>
  );
}
```

#### 4. State（状态）

State是组件内部可变的数据，当状态改变时，组件会重新渲染。

```tsx
import { useState } from 'react';

function Counter() {
  // useState返回一个状态值和一个更新函数
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```

#### 5. 生命周期与副作用

useEffect钩子用于处理副作用，如数据获取、订阅、手动修改DOM等。

```tsx
import { useState, useEffect } from 'react';

function UserProfile({ userId }: { userId: string }) {
  const [user, setUser] = useState(null);

  useEffect(() => {
    // 当userId变化时，获取用户数据
    fetchUser(userId).then(data => setUser(data));

    // 清理函数（组件卸载时执行）
    return () => {
      // 清理订阅或取消请求
    };
  }, [userId]); // 依赖数组：只有userId变化时才重新执行

  return user ? <div>{user.name}</div> : <div>Loading...</div>;
}
```

#### 6. 条件渲染

根据条件渲染不同的UI。

```tsx
function Greeting({ isLoggedIn }: { isLoggedIn: boolean }) {
  return (
    <div>
      {isLoggedIn ? (
        <p>Welcome back!</p>
      ) : (
        <p>Please log in.</p>
      )}
    </div>
  );
}

// 使用三元运算符
function Status({ status }: { status: string }) {
  return (
    <div className={status === 'success' ? 'text-green' : 'text-red'}>
      {status}
    </div>
  );
}
```

#### 7. 列表渲染

使用数组的map方法渲染列表。

```tsx
function TodoList({ items }: { items: string[] }) {
  return (
    <ul>
      {items.map((item, index) => (
        // 必须为每个列表项提供唯一的key
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}
```

#### 8. 事件处理

React使用驼峰命名法处理事件。

```tsx
function Form() {
  const [text, setText] = useState('');

  const handleSubmit = (event: React.FormEvent) => {
    event.preventDefault(); // 阻止默认行为
    console.log('Submitted:', text);
  };

  const handleChange = (event: React.ChangeEvent<HTMLInputElement>) => {
    setText(event.target.value);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input 
        type="text" 
        value={text} 
        onChange={handleChange} 
      />
      <button type="submit">Submit</button>
    </form>
  );
}
```

## Next.js框架

### 什么是Next.js？

Next.js是一个基于React的框架，提供了服务器端渲染、静态网站生成、文件系统路由等特性，用于构建高性能的Web应用。

### 核心特性

#### 1. 文件系统路由

在Next.js中，文件系统中的文件自动映射为URL路径。

```
app/
├── page.tsx        → / (根路径)
├── login/
│   └── page.tsx    → /login
├── dashboard/
│   └── page.tsx    → /dashboard
└── devices/
    └── [id]/
        └── page.tsx → /devices/:id (动态路由)
```

#### 2. 服务器组件与客户端组件

Next.js 13+引入了服务器组件和客户端组件的概念。

```tsx
// 服务器组件（默认）
// 在服务器上渲染，可以安全地访问后端资源
async function ServerComponent() {
  const data = await fetch('https://api.example.com/data');
  return <div>{data.title}</div>;
}

// 客户端组件（使用"use client"指令）
// 在浏览器中渲染，可以访问浏览器API和交互功能
"use client";
import { useState } from 'react';

function ClientComponent() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
```

#### 3. 数据获取

Next.js提供了多种数据获取方式。

```tsx
// 服务器组件中使用async/await
async function Page() {
  const data = await fetch('https://api.example.com/data');
  return <div>{data.title}</div>;
}

// 客户端组件中使用useEffect
"use client";
import { useState, useEffect } from 'react';

function Page() {
  const [data, setData] = useState(null);

  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(res => res.json())
      .then(data => setData(data));
  }, []);

  return data ? <div>{data.title}</div> : <div>Loading...</div>;
}
```

#### 4. 路由导航

Next.js提供了多种路由导航方式。

```tsx
"use client";
import { useRouter, usePathname } from 'next/navigation';

function Navigation() {
  const router = useRouter();
  const pathname = usePathname();

  const handleClick = () => {
    // 导航到新页面
    router.push('/dashboard');

    // 替换当前页面（不保留历史记录）
    router.replace('/login');
  };

  return (
    <div>
      <p>Current path: {pathname}</p>
      <button onClick={handleClick}>Go to Dashboard</button>
    </div>
  );
}
```

#### 5. 链接组件

使用Link组件进行客户端导航，避免整页刷新。

```tsx
import Link from 'next/link';

function Navigation() {
  return (
    <nav>
      <Link href="/">Home</Link>
      <Link href="/dashboard">Dashboard</Link>
      <Link href="/devices">Devices</Link>
    </nav>
  );
}
```

## TypeScript类型系统

### 什么是TypeScript？

TypeScript是JavaScript的超集，添加了静态类型定义。它使代码更易于维护，减少运行时错误。

### 基本类型

```tsx
// 基本类型
let name: string = "John";
let age: number = 30;
let isActive: boolean = true;

// 数组类型
let numbers: number[] = [1, 2, 3];
let names: string[] = ["Alice", "Bob"];

// 对象类型
interface User {
  id: string;
  name: string;
  email: string;
  age?: number; // 可选属性
}

const user: User = {
  id: "1",
  name: "John",
  email: "john@example.com"
};
```

### 函数类型

```tsx
// 函数类型
function greet(name: string): string {
  return `Hello, ${name}!`;
}

// 箭头函数类型
const add = (a: number, b: number): number => a + b;

// 使用类型别名
type EventHandler = (event: React.MouseEvent) => void;

const handleClick: EventHandler = (event) => {
  console.log('Clicked!', event);
};
```

### 泛型

泛型允许我们创建可重用的组件、函数和类。

```tsx
// 泛型函数
function identity<T>(arg: T): T {
  return arg;
}

// 泛型组件
interface ListProps<T> {
  items: T[];
  renderItem: (item: T) => React.ReactNode;
}

function List<T>({ items, renderItem }: ListProps<T>) {
  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>{renderItem(item)}</li>
      ))}
    </ul>
  );
}

// 使用泛型组件
<List 
  items={['Apple', 'Banana', 'Cherry']} 
  renderItem={(item) => <span>{item}</span>} 
/>
```

### 类型断言

```tsx
// 类型断言
let value: any = "Hello, world!";
let length: number = (value as string).length;

// 非空断言
function processInput(input: string | null) {
  // 如果确定input不为null，可以使用非空断言
  console.log(input!.toUpperCase());
}
```

### 什么是Ant Design？

Ant Design（简称antd）是一套企业级UI设计语言和React UI组件库，提供了一套完整的设计规范和高质量的React组件。


### 组件组织原则

1. **单一职责**：每个组件只负责一个功能
2. **可复用性**：将通用功能提取为独立组件
3. **组件组合**：通过组合简单组件构建复杂UI
4. **关注点分离**：将状态管理、副作用与UI分离

## 核心概念与模式

### 1. Context API

Context提供了一种在组件树中共享数据的方式，避免通过props层层传递。

```tsx
import { createContext, useContext, useState } from 'react';

// 创建Context
interface AuthContextType {
  isAuthenticated: boolean;
  login: (username: string, password: string) => void;
  logout: () => void;
}

const AuthContext = createContext<AuthContextType | undefined>(undefined);

// 创建Provider组件
function AuthProvider({ children }: { children: React.ReactNode }) {
  const [isAuthenticated, setIsAuthenticated] = useState(false);

  const login = (username: string, password: string) => {
    // 登录逻辑
    setIsAuthenticated(true);
  };

  const logout = () => {
    // 登出逻辑
    setIsAuthenticated(false);
  };

  return (
    <AuthContext.Provider value={{ isAuthenticated, login, logout }}>
      {children}
    </AuthContext.Provider>
  );
}

// 创建自定义Hook使用Context
function useAuth() {
  const context = useContext(AuthContext);
  if (!context) {
    throw new Error('useAuth must be used within AuthProvider');
  }
  return context;
}

// 在组件中使用
function LoginPage() {
  const { login } = useAuth();

  return <button onClick={() => login('user', 'pass')}>Login</button>;
}
```

### 2. 自定义Hooks

自定义Hooks允许我们将组件逻辑提取到可复用的函数中。

```tsx
import { useState, useEffect } from 'react';

// 自定义Hook：获取窗口大小
function useWindowSize() {
  const [size, setSize] = useState({
    width: window.innerWidth,
    height: window.innerHeight
  });

  useEffect(() => {
    const handleResize = () => {
      setSize({
        width: window.innerWidth,
        height: window.innerHeight
      });
    };

    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return size;
}

// 使用自定义Hook
function ResponsiveComponent() {
  const { width, height } = useWindowSize();

  return (
    <div>
      Window size: {width} x {height}
    </div>
  );
}
```

### 3. 受控组件与非受控组件

受控组件的值由React状态控制，非受控组件的值由DOM控制。

```tsx
// 受控组件
function ControlledInput() {
  const [value, setValue] = useState('');

  return (
    <input 
      type="text" 
      value={value} 
      onChange={(e) => setValue(e.target.value)} 
    />
  );
}

// 非受控组件
function UncontrolledInput() {
  const inputRef = useRef<HTMLInputElement>(null);

  const getValue = () => {
    if (inputRef.current) {
      return inputRef.current.value;
    }
    return '';
  };

  return <input type="text" ref={inputRef} defaultValue="Initial value" />;
}
```

### 4. 性能优化

使用React提供的优化技术提高应用性能。

```tsx
import { useState, useMemo, useCallback } from 'react';

function ExpensiveComponent({ items, onClick }: { 
  items: number[]; 
  onClick: (item: number) => void 
}) {
  // 使用useMemo缓存计算结果
  const sortedItems = useMemo(() => {
    return [...items].sort((a, b) => a - b);
  }, [items]);

  // 使用useCallback缓存函数引用
  const handleClick = useCallback((item: number) => {
    onClick(item);
  }, [onClick]);

  return (
    <ul>
      {sortedItems.map(item => (
        <li key={item} onClick={() => handleClick(item)}>
          {item}
        </li>
      ))}
    </ul>
  );
}
```

### 5. 错误处理

处理组件中的错误，防止应用崩溃。

```tsx
import { Component, ReactNode } from 'react';

interface ErrorBoundaryState {
  hasError: boolean;
}

class ErrorBoundary extends Component<
  { children: ReactNode },
  ErrorBoundaryState
> {
  state = { hasError: false };

  static getDerivedStateFromError(error: Error) {
    return { hasError: true };
  }

  componentDidCatch(error: Error, errorInfo: any) {
    console.error('Error caught by boundary:', error, errorInfo);
  }

  render() {
    if (this.state.hasError) {
      return <div>Something went wrong.</div>;
    }
    return this.props.children;
  }
}

// 使用错误边界
<ErrorBoundary>
  <MyComponent />
</ErrorBoundary>
```
