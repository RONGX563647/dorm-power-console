```java
@Configuration
@EnableWebSecurity
@EnableMethodSecurity(prePostEnabled = true)
public class SecurityConfig {

    @Autowired
    private JwtAuthenticationFilter jwtAuthenticationFilter;

    @Bean
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
            .cors(cors -> cors.configurationSource(corsConfigurationSource))
            .csrf(AbstractHttpConfigurer::disable)  // JWT无状态应用禁用CSRF
            .authorizeHttpRequests(authorize -> authorize

                // 需要认证的路径

                // 其他请求需要认证
                .anyRequest().authenticated()
            )
            .addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class);
        return http.build();
    }
}
```

### 2. JWT认证过滤器

#### JwtAuthenticationFilter.java

```java
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    @Autowired
    private JwtUtil jwtUtil;

    @Value("${security.jwt.secret}")
    private String secret;

    @Override
    protected void doFilterInternal(
        HttpServletRequest request,
        HttpServletResponse response,
        FilterChain filterChain
    ) throws ServletException, IOException {
        String token = getTokenFromRequest(request);

        if (token != null && jwtUtil.validateToken(token)) {
            try {
                Claims claims = Jwts.parserBuilder()
                        .setSigningKey(getSigningKey())
                        .build()
                        .parseClaimsJws(token)
                        .getBody();

                String username = claims.getSubject();
                String role = claims.get("role", String.class);

                // 获取用户权限
                List<SimpleGrantedAuthority> authorities = getUserAuthorities(username, role);

                // 创建认证对象
                UsernamePasswordAuthenticationToken authentication =
                    new UsernamePasswordAuthenticationToken(username, null, authorities);
                authentication.setDetails(new WebAuthenticationDetailsSource().buildDetails(request));

                // 设置到SecurityContext
                SecurityContextHolder.getContext().setAuthentication(authentication);
            } catch (Exception e) {
                // 令牌无效，继续过滤链
            }
        }

        filterChain.doFilter(request, response);
    }
}
```

**核心流程**:
1. 从请求头提取Bearer Token
2. 解析JWT并验证签名
3. 从RBAC系统获取用户权限
4. 创建`Authentication`对象
5. 设置到`SecurityContext`

### 3. AOP切面 - 统一日志与限流

#### ApiAspect.java

```java
@Aspect
@Component
public class ApiAspect {

    @Autowired
    private RateLimiter apiRateLimiter;

    private final Logger logger = LoggerFactory.getLogger(ApiAspect.class);
    private final Logger auditLogger = LoggerFactory.getLogger("AUDIT");

    /**
     * API请求日志和限流处理
     */
    @Around("@annotation(com.dormpower.annotation.RateLimit) || " +
            "@within(org.springframework.web.bind.annotation.RestController)")
    public Object handleApiRequest(ProceedingJoinPoint joinPoint) throws Throwable {
        long startTime = System.currentTimeMillis();

        HttpServletRequest request = getRequest();
        String method = request.getMethod();
        String uri = request.getRequestURI();
        String ip = getClientIp(request);

        // 限流检查
        if (!isAdminUser()) {
            RateLimit rateLimit = targetMethod.getAnnotation(RateLimit.class);
            if (rateLimit != null) {
                RateLimiter limiter = getRateLimiter(rateLimit);
                if (!limiter.tryAcquire()) {
                    throw new RuntimeException("请求过于频繁，请稍后再试");
                }
            } else {
                if (!apiRateLimiter.tryAcquire()) {
                    throw new RuntimeException("请求过于频繁，请稍后再试");
                }
            }
        }

        logger.debug("请求开始: {} {} from {}", method, uri, ip);

        try {
            Object result = joinPoint.proceed();  // 执行目标方法

            long duration = System.currentTimeMillis() - startTime;
            logger.info("请求完成: {} {} - {}ms - {}", method, uri, duration, ip);

            return result;
        } catch (Throwable e) {
            long duration = System.currentTimeMillis() - startTime;
            logger.error("请求失败: {} {} - {}ms - {} - {}",
                method, uri, duration, ip, e.getMessage());
            throw e;
        }
    }
}
```

#### AOP切点说明

| 切点表达式 | 说明 |
|-----------|------|
| `@annotation(RateLimit)` | 匹配有`@RateLimit`注解的方法 |
| `@within(RestController)` | 匹配所有`@RestController`注解的类 |

**AOP执行流程**:
```
请求到达 → AOP切面拦截 → 限流检查 → 记录开始日志 →
执行业务方法 → 记录结束日志 → 返回响应
          ↓ 异常发生
    记录错误日志 → 抛出异常
```

#### Guava RateLimiter 工作原理

**令牌桶算法**:
- 桶容量 = 速率 × 时间
- 每秒向桶中放入令牌
- 请求到来时，尝试获取令牌
- 有令牌则放行，否则阻塞或拒绝

**使用示例**:
```java
if (apiRateLimiter.tryAcquire()) {
    // 处理请求
} else {
    // 限流，拒绝请求
}
```

#### MqttConfig.java

```java
@Configuration
public class MqttConfig {

    @Value("${mqtt.enabled:false}")
    private boolean enabled;

    @Value("${mqtt.broker-url}")
    private String brokerUrl;

    public MqttClient createMqttClient() {
        if (!enabled) {
            return null;
        }

        try {
            MqttClient client = new MqttClient(brokerUrl, clientId);

            MqttConnectOptions options = new MqttConnectOptions();
            options.setAutomaticReconnect(true);  // 自动重连
            options.setCleanSession(true);        // 清除会话
            options.setConnectionTimeout(10);     // 连接超时
            options.setKeepAliveInterval(60);     // 心跳间隔

            client.connect(options);
            return client;
        } catch (MqttException e) {
            return null;
        }
    }
}
```

#### MqttService.java

```java
@Service
public class MqttService {

    private MqttClient mqttClient;
    private BiConsumer<String, String> messageHandler;

    /**
     * 发送MQTT消息
     */
    public void sendMessage(String topic, String message) throws MqttException {
        if (mqttClient != null && mqttClient.isConnected()) {
            MqttMessage mqttMessage = new MqttMessage(message.getBytes());
            mqttMessage.setQos(1);  // 至少一次送达
            mqttClient.publish(topic, mqttMessage);
        }
    }

    /**
     * 订阅MQTT主题
     */
    public void subscribe(String topic) throws MqttException {
        if (mqttClient != null && mqttClient.isConnected()) {
            mqttClient.subscribe(topic, (topic1, message) -> {
                String payload = new String(message.getPayload());
                if (messageHandler != null) {
                    messageHandler.accept(topic1, payload);
                }
            });
        }
    }
}
```

### 1. 认证服务 (AuthService)

#### 登录流程

```java
@Service
public class AuthService {

    @Autowired
    private UserAccountRepository userAccountRepository;

    @Autowired
    private JwtUtil jwtUtil;

    public Map<String, Object> login(String account, String password) {
        String normalizedAccount = account.trim();
        String normalizedAdminUsername = adminUsername.trim();
        String normalizedAdminEmail = adminEmail.trim().toLowerCase();

        // 1. 验证账号格式
        if (!normalizedAdminUsername.equals(normalizedAccount) &&
            !normalizedAdminEmail.equals(normalizedAccount.toLowerCase())) {
            throw new AuthenticationException("Invalid credentials");
        }

        // 2. 查询用户
        UserAccount user = userAccountRepository.findById(normalizedAdminUsername).orElse(null);
        if (user == null) {
            throw new ResourceNotFoundException("User not found");
        }

        // 3. 验证密码 (PBKDF2哈希)
        if (!EncryptionUtil.verifyPassword(password, user.getPasswordHash())) {
            throw new AuthenticationException("Invalid credentials");
        }

        // 4. 生成JWT令牌
        String token = jwtUtil.generateToken(user.getUsername(), user.getRole());

        // 5. 构造响应
        Map<String, Object> response = new HashMap<>();
        response.put("token", token);
        response.put("user", Map.of(
            "username", user.getUsername(),
            "email", user.getEmail(),
            "role", user.getRole()
        ));
        return response;
    }
}
```

#### 密码安全机制

```java
public static String hashPassword(String password) {
    String salt = generateSalt();
    String digest = hashPassword(password, salt, ITERATIONS);
    return String.format("pbkdf2_sha256$%d$%s$%s", ITERATIONS, salt, digest);
}

public static boolean verifyPassword(String password, String encoded) {
    String[] parts = encoded.split("\\$");
    String salt = parts[2];
    String expectedDigest = parts[3];
    String actualDigest = hashPassword(password, salt, iterations);

    // 恒定时间比较，防止时序攻击
    return constantTimeEquals(expectedDigest, actualDigest);
}
```

**安全特性**:
- 使用PBKDF2WithHmacSHA256算法
- 160000次迭代
- 随机盐值
- 恒定时间比较防止时序攻击

### 2. AI分析报告服务 (AiReportService)

#### 核心算法

```java
@Service
public class AiReportService {

    @Autowired
    private DeviceRepository deviceRepository;

    @Autowired
    private TelemetryRepository telemetryRepository;

    public Map<String, Object> getAiReport(String roomId, String period) {
        // 1. 查询房间设备
        List<Device> devices = deviceRepository.findByRoom(roomId);
        if (devices.isEmpty()) {
            return createEmptyReport(roomId, "No device data in this room yet.", ...);
        }

        // 2. 计算时间范围
        final int days = "7d".equals(period) ? 7 : 30;
        final long nowTs = System.currentTimeMillis() / 1000;
        final long startTs = nowTs - days * 24 * 3600L;

        // 3. 获取遥测数据
        List<Telemetry> allData = new ArrayList<>();
        for (Device d : devices) {
            List<Telemetry> deviceData = telemetryRepository.findByDeviceIdAndTsBetweenOrderByTsAsc(d.getId(), startTs, nowTs);
            allData.addAll(deviceData);
        }

        if (allData.isEmpty()) {
            return createEmptyReport(roomId, "Not enough telemetry points.", ...);
        }

        return analyzeData(roomId, allData);
    }

    private Map<String, Object> analyzeData(String roomId, List<Telemetry> data) {
        // 4. 计算统计指标
        double sumPower = 0.0;
        double peakPower = 0.0;
        long peakTs = 0;

        for (Telemetry t : data) {
            double power = t.getPowerW();
            sumPower += power;

            if (power > peakPower) {
                peakPower = power;
                peakTs = t.getTs();
            }
        }

        double avgPower = sumPower / data.size();
        double totalKwh = (avgPower * dataSize * 60) / (1000.0 * 3600.0);

        // 5. 异常检测
        List<String> anomalies = new ArrayList<>();
        if (peakPower > 200) {
            anomalies.add(String.format("Peak power reached %.1fW. Check high-load periods.", peakPower));
        }
        if (avgPower > 150) {
            anomalies.add(String.format("Average power is high at %.1fW.", avgPower));
        }

        // 6. 智能建议
        List<String> recommendations = new ArrayList<>();
        recommendations.add("Enable auto off for low-priority sockets after 00:30.");
        if (avgPower > 100) {
            recommendations.add("Schedule high-power devices during off-peak hours.");
        }

        return createReport(roomId, avgPower, peakPower, anomalies, recommendations);
    }
}
```

```java
public class DeviceService {

    private DeviceRepository deviceRepository;

     * 控制设备状态
     */

        DeviceStatusHistory history = new DeviceStatusHistory();
        history.setDeviceId(deviceId);
        history.setCommand(command);
        history.setStatus("PENDING");

     * 查询设备状态
     */
    public DeviceStatus getDeviceStatus(String deviceId) {

            .orElseThrow(() -> new ResourceNotFoundException("Device not found"));

            .findTopByDeviceIdOrderByTsDesc(deviceId);

        return DeviceStatus.builder()
            .deviceId(deviceId)
            .status(latestStatus != null ? latestStatus.getStrips() : null)
            .lastActive(latestStatus != null ? latestStatus.getTs() : 0)
            .build();
    }
```

### 1. JWT令牌认证

#### 生成令牌

```java
public String generateToken(String username, String role) {
    Date now = new Date();
    Date expiryDate = new Date(now.getTime() + expiration);

    return Jwts.builder()
            .setSubject(username)
            .claim("username", username)
            .claim("role", role)
            .setIssuedAt(now)
            .setExpiration(expiryDate)
            .signWith(getSigningKey())  // HMAC-SHA256签名
            .compact();
}
```

#### 验证令牌

```java
public boolean validateToken(String token) {
    // 1. 检查黑名单
    if (tokenBlacklist.isBlacklisted(token)) {
        return false;
    }

    try {
        // 2. 解析并验证签名
        Jwts.parserBuilder()
                .setSigningKey(getSigningKey())
                .build()
                .parseClaimsJws(token);
        return true;
    } catch (Exception e) {
        return false;
    }
}
```

#### 令牌黑名单 (登出)

```java
@Component
public class TokenBlacklist {

    // 使用ConcurrentHashMap存储已登出的令牌
    private final ConcurrentHashMap<String, Long> blacklist = new ConcurrentHashMap<>();

    // 令牌有效期（2小时）
    private static final long EXPIRATION_MS = 2 * 60 * 60 * 1000;

    public void addToBlacklist(String token) {
        // 将令牌加入黑名单，记录时间戳
        blacklist.put(token, System.currentTimeMillis());

        // 清理过期的黑名单条目
        cleanupExpiredTokens();
    }

    public boolean isBlacklisted(String token) {
        Long timestamp = blacklist.get(token);
        if (timestamp == null) {
            return false;
        }

        // 检查是否已过期
        return System.currentTimeMillis() - timestamp < EXPIRATION_MS;
    }

    private void cleanupExpiredTokens() {
        blacklist.entrySet().removeIf(entry ->
            System.currentTimeMillis() - entry.getValue() > EXPIRATION_MS
        );
    }
}
```

### 2. 密码加密存储

#### PBKDF2算法

```java
public static String hashPassword(String password) {
    String salt = generateSalt();  // 16字节随机盐
    String digest = hashPassword(password, salt, 160000);  // 160000次迭代
    return String.format("pbkdf2_sha256$%d$%s$%s", 160000, salt, digest);
}

private static String hashPassword(String password, String salt, int iterations) {
    PBEKeySpec spec = new PBEKeySpec(
        password.toCharArray(),
        HexFormat.of().parseHex(salt),
        iterations,
        256  // 密钥长度256位
    );
    SecretKeyFactory factory = SecretKeyFactory.getInstance("PBKDF2WithHmacSHA256");
    byte[] hash = factory.generateSecret(spec).getEncoded();
    return HexFormat.of().formatHex(hash);
}
```

**安全优势**:
- **盐值 (Salt)**: 防止彩虹表攻击
- **迭代次数**: 增加破解成本
- **算法**: PBKDF2是行业标准

```java
// 错误示例 - 容易受到时序攻击
public boolean verifyPassword(String password, String encoded) {
    String[] parts = encoded.split("\\$");
    String actualDigest = hashPassword(password, parts[2], Integer.parseInt(parts[1]));
    return actualDigest.equals(parts[3]);  // 字符串比较时间不同
}

// 正确示例 - 恒定时间比较
private static boolean constantTimeEquals(String a, String b) {
    if (a.length() != b.length()) {
        return false;
    }
    int result = 0;
    for (int i = 0; i < a.length(); i++) {
        result |= a.charAt(i) ^ b.charAt(i);  // 按位异或
    }
    return result == 0;
}
```
