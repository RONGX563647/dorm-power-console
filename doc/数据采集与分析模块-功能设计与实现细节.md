# 数据采集与分析模块 - 功能设计与实现细节文档

## 1. 模块概述

### 1.1 模块定位

数据采集与分析模块是宿舍电源管理系统的核心模块之一，负责采集设备用电数据并提供多维度分析能力。该模块提供遥测数据采集、查询、采样聚合、AI分析报告和异常检测等功能，为系统提供数据支撑和智能分析能力。

#### 2.1.6 实现代码

**MQTT消息处理：**

```java
@Component
public class MqttBridge {

    @Autowired
    private TelemetryRepository telemetryRepository;

    @Autowired(required = false)
    private TelemetryProducer telemetryProducer;

    @Value("${kafka.enabled:true}")
    private boolean kafkaEnabled;

    private void handleMessage(String topic, String payload) {
        try {
            JsonNode root = objectMapper.readTree(payload);

            if (topic.endsWith("/telemetry")) {
                handleTelemetryMessage(topic, root);
            }
        } catch (Exception e) {
            logger.error("处理MQTT消息失败: {}", e.getMessage());
        }
    }

    private void handleTelemetryMessage(String topic, JsonNode root) {
        String deviceId = extractDeviceId(topic);
        long ts = root.has("ts") ? root.get("ts").asLong() : System.currentTimeMillis() / 1000;
        double powerW = root.has("power_w") ? root.get("power_w").asDouble() : 0;
        double voltageV = root.has("voltage_v") ? root.get("voltage_v").asDouble() : 220;
        double currentA = root.has("current_a") ? root.get("current_a").asDouble() : 0;

        if (kafkaEnabled && telemetryProducer != null) {
            telemetryProducer.sendTelemetry(deviceId, ts, powerW, voltageV, currentA);
        } else {
            saveTelemetryDirectly(deviceId, ts, powerW, voltageV, currentA);
        }
    }

    private void saveTelemetryDirectly(String deviceId, long ts, 
            double powerW, double voltageV, double currentA) {
        Telemetry telemetry = new Telemetry();
        telemetry.setDeviceId(deviceId);
        telemetry.setTs(ts);
        telemetry.setPowerW(powerW);
        telemetry.setVoltageV(voltageV);
        telemetry.setCurrentA(currentA);
        telemetryRepository.save(telemetry);
    }
}
```

```java
@Service
public class TelemetryService {

    private static final Map<String, RangeConfig> RANGE_CONFIG;
    
    static {
        Map<String, RangeConfig> config = new HashMap<>(4);
        config.put("60s", new RangeConfig(60, 1));
        config.put("24h", new RangeConfig(96, 900));
        config.put("7d", new RangeConfig(168, 3600));
        config.put("30d", new RangeConfig(120, 21600));
        RANGE_CONFIG = Collections.unmodifiableMap(config);
    }

    @Cacheable(value = "telemetry", key = "#deviceId + '_' + #range")
    public List<Map<String, Object>> getTelemetry(String deviceId, String range) {
        RangeConfig config = RANGE_CONFIG.get(range);
        if (config == null) {
            throw new BusinessException("Invalid range: " + range);
        }

        final int points = config.points;
        final int step = config.step;
        final long nowTs = System.currentTimeMillis() / 1000;
        final long startTs = nowTs - (long) (points - 1) * step;

        List<Telemetry> rows = telemetryRepository
            .findByDeviceIdAndTsBetweenOrderByTsAsc(deviceId, startTs, nowTs);

        if (rows.isEmpty()) {
            if (range.equals("60s")) {
                return generateMockData(points, step, startTs);
            }
            return Collections.emptyList();
        }

        return processTelemetryData(deviceId, rows, points, step, startTs, range);
    }

    private static final class RangeConfig {
        final int points;
        final int step;
        RangeConfig(int points, int step) {
            this.points = points;
            this.step = step;
        }
    }
}
```

#### 2.3.4 实现代码

**槽位填充算法（60秒范围）：**

```java
private List<Map<String, Object>> process60sRange(String deviceId, 
        List<Telemetry> rows, int points, int step, long startTs) {
    
    List<Map<String, Object>> result = new ArrayList<>(points);
    
    // 获取起始时间之前的最后一个数据点作为初始值
    Telemetry prevRow = telemetryRepository
        .findFirstByDeviceIdAndTsLessThanOrderByTsDesc(deviceId, startTs);
    double prevPower = prevRow != null ? prevRow.getPowerW() : 0.0;

    final int rowsSize = rows.size();
    int rowIdx = 0;
    
    // 按时间槽位填充数据
    for (int i = 0; i < points; i++) {
        long slotTs = startTs + (long) i * step;
        double power = prevPower;

        // 找到该槽位最新的数据点
        while (rowIdx < rowsSize) {
            Telemetry row = rows.get(rowIdx);
            if (row.getTs() <= slotTs) {
                power = row.getPowerW();
                rowIdx++;
            } else {
                break;
            }
        }

        Map<String, Object> data = new HashMap<>(2);
        data.put("ts", slotTs);
        data.put("power_w", roundPower(power));
        result.add(data);
    }

    return result;
}
```

**数据采样算法（非60秒范围）：**

```java
private List<Map<String, Object>> processNon60sRange(List<Telemetry> rows, int points) {
    final int size = rows.size();
    
    // 数据点数少于目标点数，直接返回
    if (size <= points) {
        List<Map<String, Object>> result = new ArrayList<>(size);
        for (Telemetry r : rows) {
            Map<String, Object> data = new HashMap<>(2);
            data.put("ts", r.getTs());
            data.put("power_w", roundPower(r.getPowerW()));
            result.add(data);
        }
        return result;
    }

    // 均匀采样
    List<Map<String, Object>> result = new ArrayList<>(points);
    final double stepIdx = (double) (size - 1) / (points - 1);
    
    for (int i = 0; i < points; i++) {
        int idx = (int) Math.round(i * stepIdx);
        Telemetry r = rows.get(idx);
        Map<String, Object> data = new HashMap<>(2);
        data.put("ts", r.getTs());
        data.put("power_w", roundPower(r.getPowerW()));
        result.add(data);
    }
    return result;
}

private static double roundPower(double power) {
    return Math.round(power * 1000.0) / 1000.0;
}
```

```java
@Service
public class AiReportService {

    @Autowired
    private DeviceRepository deviceRepository;

    @Autowired
    private TelemetryRepository telemetryRepository;

    public Map<String, Object> getAiReport(String roomId, String period) {
        List<Device> devices = deviceRepository.findByRoom(roomId);
        
        if (devices.isEmpty()) {
            return createEmptyReport(roomId, 
                "No device data in this room yet.",
                Collections.singletonList("No analyzable sample found."),
                Collections.singletonList("Ensure devices upload status and telemetry periodically.")
            );
        }

        final int days = "7d".equals(period) ? 7 : 30;
        final long nowTs = System.currentTimeMillis() / 1000;
        final long startTs = nowTs - days * 24 * 3600L;

        List<Telemetry> allData = new ArrayList<>();
        for (Device d : devices) {
            List<Telemetry> deviceData = telemetryRepository
                .findByDeviceIdAndTsBetweenOrderByTsAsc(d.getId(), startTs, nowTs);
            allData.addAll(deviceData);
        }

        if (allData.isEmpty()) {
            return createEmptyReport(roomId,
                "Devices are online but telemetry coverage is insufficient.",
                Collections.singletonList("Not enough telemetry points in selected period."),
                Collections.singletonList("Increase telemetry frequency to every 1-5 seconds.")
            );
        }

        return analyzeData(roomId, allData);
    }

    private Map<String, Object> analyzeData(String roomId, List<Telemetry> data) {
        final int dataSize = data.size();
        
        double sumPower = 0.0;
        double peakPower = 0.0;
        long peakTs = 0;
        
        for (Telemetry t : data) {
            double power = t.getPowerW();
            sumPower += power;
            if (power > peakPower) {
                peakPower = power;
                peakTs = t.getTs();
            }
        }
        
        final double avgPower = sumPower / dataSize;
        final String peakTime = peakTs > 0 ? 
            Instant.ofEpochSecond(peakTs).toString() : "";
        final double totalKwh = (avgPower * dataSize * 60) / (1000.0 * 3600.0);

        // 异常检测
        List<String> anomalies = new ArrayList<>();
        if (peakPower > 200) {
            anomalies.add(String.format(
                "Peak power reached %.1fW. Check high-load periods.", peakPower));
        }
        if (avgPower > 150) {
            anomalies.add(String.format(
                "Average power is high at %.1fW. Consider reducing load.", avgPower));
        }

        // 生成建议
        List<String> recommendations = new ArrayList<>();
        recommendations.add("Enable auto off for low-priority sockets after 00:30.");
        recommendations.add("Set alerts for periods above baseline by 20%.");
        if (avgPower > 100) {
            recommendations.add("Consider scheduling high-power devices during off-peak hours.");
        }

        Map<String, Object> report = new HashMap<>();
        report.put("room_id", roomId);
        report.put("summary", String.format(
            "Average power is about %.1fW, peak is about %.1fW.", avgPower, peakPower));
        report.put("anomalies", anomalies);
        report.put("recommendations", recommendations);
        report.put("generated_at", System.currentTimeMillis() / 1000);
        report.put("power_stats", createPowerStats(avgPower, peakPower, peakTime, totalKwh));
        
        return report;
    }
}
```

#### 2.5.4 实现代码

**异常检测Agent：**

```java
@Service
public class AnomalyDetectionAgent {

    private static final int WINDOW_SIZE = 30;
    private static final double Z_SCORE_THRESHOLD = 3.0;
    private static final double EWMA_ALPHA = 0.3;
    private static final int MIN_SAMPLES = 10;

    public enum AnomalyType {
        POINT_ANOMALY,           // 点异常
        CONTEXTUAL_ANOMALY,      // 上下文异常
        TREND_ANOMALY,           // 趋势异常
        COLLECTIVE_ANOMALY,      // 集合异常
        PREDICTIVE_ANOMALY       // 预测性异常
    }

    public enum Severity {
        LOW, MEDIUM, HIGH, CRITICAL
    }

    /**
     * 实时异常检测
     */
    public DetectedAnomaly detectRealtime(String deviceId, double currentPower) {
        Device device = deviceRepository.findById(deviceId).orElse(null);
        if (device == null) return null;

        long now = System.currentTimeMillis() / 1000;
        long startTs = now - 3600;

        List<Telemetry> recentData = telemetryRepository
            .findByDeviceIdAndTsBetweenOrderByTsAsc(deviceId, startTs, now);

        if (recentData.size() < MIN_SAMPLES) return null;

        List<Double> powers = recentData.stream()
            .map(Telemetry::getPowerW)
            .collect(Collectors.toList());

        double mean = MLUtils.mean(powers);
        double std = MLUtils.stdDev(powers);

        if (MLUtils.isAnomalyByZScore(currentPower, mean, std, Z_SCORE_THRESHOLD)) {
            DetectedAnomaly anomaly = new DetectedAnomaly();
            anomaly.anomalyId = "anomaly_" + System.currentTimeMillis();
            anomaly.deviceId = deviceId;
            anomaly.roomId = device.getRoom();
            anomaly.type = AnomalyType.POINT_ANOMALY;
            anomaly.value = currentPower;
            anomaly.expectedValue = mean;
            anomaly.deviationScore = Math.abs((currentPower - mean) / (std > 0 ? std : 1));
            anomaly.timestamp = now;
            anomaly.recommendations = new ArrayList<>();

            if (currentPower > mean + 3 * std) {
                anomaly.severity = Severity.HIGH;
                anomaly.description = String.format(
                    "功率异常升高: 当前 %.1fW, 预期 %.1fW", currentPower, mean);
                anomaly.recommendations.add("检查是否有大功率设备接入");
            } else {
                anomaly.severity = Severity.MEDIUM;
                anomaly.description = String.format(
                    "功率异常降低: 当前 %.1fW, 预期 %.1fW", currentPower, mean);
                anomaly.recommendations.add("检查设备是否正常工作");
            }

            return anomaly;
        }

        return null;
    }

    /**
     * 批量异常检测
     */
    public List<DetectedAnomaly> detectBatch(String deviceId, long startTs, long endTs) {
        List<DetectedAnomaly> anomalies = new ArrayList<>();

        Device device = deviceRepository.findById(deviceId).orElse(null);
        if (device == null) return anomalies;

        List<Telemetry> data = telemetryRepository
            .findByDeviceIdAndTsBetweenOrderByTsAsc(deviceId, startTs, endTs);

        if (data.size() < MIN_SAMPLES) return anomalies;

        List<Double> powers = data.stream()
            .map(Telemetry::getPowerW)
            .collect(Collectors.toList());

        // 检测各类异常
        anomalies.addAll(detectPointAnomalies(device, data, powers));
        anomalies.addAll(detectTrendAnomalies(device, data, powers));
        anomalies.addAll(detectContextualAnomalies(device, data, powers));

        return anomalies;
    }

    /**
     * 获取设备健康状态
     */
    public DeviceHealthStatus getDeviceHealth(String deviceId) {
        Device device = deviceRepository.findById(deviceId).orElse(null);
        if (device == null) return null;

        long now = System.currentTimeMillis() / 1000;
        long startTs = now - 86400;

        List<Telemetry> data = telemetryRepository
            .findByDeviceIdAndTsBetweenOrderByTsAsc(deviceId, startTs, now);

        DeviceHealthStatus status = new DeviceHealthStatus();
        status.deviceId = deviceId;
        status.deviceName = device.getName();
        status.lastChecked = now;

        if (data.size() < MIN_SAMPLES) {
            status.healthScore = 50;
            status.status = "数据不足";
            return status;
        }

        List<Double> powers = data.stream()
            .map(Telemetry::getPowerW)
            .collect(Collectors.toList());

        double mean = MLUtils.mean(powers);
        double std = MLUtils.stdDev(powers);

        // 计算健康分数
        double stabilityScore = 100.0;
        double cv = std / (mean > 0 ? mean : 1);
        if (cv > 0.5) {
            stabilityScore -= 20;
        }

        List<DetectedAnomaly> anomalies = detectBatch(deviceId, startTs, now);
        long criticalCount = anomalies.stream()
            .filter(a -> a.severity == Severity.CRITICAL).count();
        long highCount = anomalies.stream()
            .filter(a -> a.severity == Severity.HIGH).count();

        double anomalyPenalty = Math.min(criticalCount * 5 + highCount * 3, 40);
        double healthScore = stabilityScore * 0.6 + (100 - anomalyPenalty) * 0.4;

        status.healthScore = Math.round(healthScore * 10.0) / 10.0;
        status.status = healthScore >= 80 ? "健康" : 
                       healthScore >= 60 ? "良好" : 
                       healthScore >= 40 ? "一般" : "异常";

        return status;
    }
}
```

**ML工具类：**

```java
public class MLUtils {

    public static double mean(List<Double> values) {
        return values.stream()
            .mapToDouble(Double::doubleValue)
            .average()
            .orElse(0.0);
    }

    public static double stdDev(List<Double> values) {
        double mean = mean(values);
        double variance = values.stream()
            .mapToDouble(v -> Math.pow(v - mean, 2))
            .average()
            .orElse(0.0);
        return Math.sqrt(variance);
    }

    public static boolean isAnomalyByZScore(double value, double mean, 
            double std, double threshold) {
        if (std == 0) return false;
        double zScore = Math.abs((value - mean) / std);
        return zScore > threshold;
    }

    public static List<Double> movingAverage(List<Double> values, int window) {
        List<Double> result = new ArrayList<>();
        for (int i = window - 1; i < values.size(); i++) {
            double sum = 0;
            for (int j = i - window + 1; j <= i; j++) {
                sum += values.get(j);
            }
            result.add(sum / window);
        }
        return result;
    }

    public static double ewmaPredict(List<Double> values, double alpha) {
        double forecast = values.get(0);
        for (int i = 1; i < values.size(); i++) {
            forecast = alpha * values.get(i) + (1 - alpha) * forecast;
        }
        return forecast;
    }
}
```

### 7.1 数据库优化

1. **索引优化**

   - 定期分析索引使用情况

   ```sql
   -- 按时间分区
   CREATE TABLE telemetry_y2024m03 PARTITION OF telemetry
   FOR VALUES FROM ('2024-03-01') TO ('2024-04-01');
   ```

   - 定期清理超过30天的数据
   - 可选：将历史数据迁移到时序数据库（如InfluxDB）

### 7.2 查询优化

1. **采样聚合**
   - 使用预聚合表减少实时计算
   - 对长时间范围查询使用降采样

2. **缓存策略**
   - 热点数据缓存
   - 查询结果缓存
